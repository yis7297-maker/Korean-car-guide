## 차량 재선택 시 기능 목록 접힘 처리 완료 여부

완료.

- 같은 차량을 다시 클릭하면 `state.vehicleId`가 해제되어 해당 차량 기능 목록이 접힙니다.
- 다른 차량을 클릭하면 기존 선택은 닫히고 새 차량만 활성화됩니다.
- 선택된 차량은 `.is-active` 상태로 시각 표시됩니다.
- 모바일에서는 차량 선택 후 기능 목록 위치로 스크롤되도록 처리했습니다.

## 기능 팝업 내 “적용 브랜드/차종/연식/트림” 항목 삭제 여부

완료.

- 기능 상세 팝업에서 기존 `적용 브랜드/차종/연식/트림` 장문 목록을 제거했습니다.
- 검색 결과에서 기능을 바로 클릭하면 `적용 여부 안내`만 표시합니다.
- 차량 선택 상태에서 기능을 클릭하면 해당 선택 차량의 적용 정보만 `선택 차량 적용 정보`로 짧게 표시합니다.

## 차량 선택 화면에서만 적용 연식/트림/옵션 정보가 표시되는지 여부

완료.

- 차량별 기능 목록 카드 안에만 적용 연식, 적용 트림, 기본/선택 옵션 여부, 기능 적용 출처를 표시합니다.
- 기능 상세 팝업은 기능 자체의 설명, 사용 방법, 작동 조건, 미작동 가능 상황, 주의/경고 사항, 관련 기능, 공식 출처 중심으로 구성했습니다.

## 추가 검수 메모

- 상용차/버스 제외 후 차량 수는 35개입니다.
- 기능 수는 11개입니다.
- 기능-차량 매핑 수는 상용차 제외 기준으로 재계산되어 표시됩니다.
- 플레오스 3개 기능은 공식 세부 카탈로그/가격표 원문 재확인 전까지 검증 미완료 상태를 유지합니다.
