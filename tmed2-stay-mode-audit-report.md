- 스테이 모드 관련 검색 키워드: 18개 반영
- 하이브리드 전용 기능 추가 수: 10개
- 관련 기능 링크 연결 여부: 완료
- 공식 출처 연결 여부: 완료
- 업데이트 로그 반영 여부: 완료

## 스테이 모드 적용 차량

공식 공개 범위를 보수적으로 적용해 다음 차량만 매핑했습니다.

- 현대 팰리세이드 하이브리드 — TMED-2

세부 적용 연식, 트림과 옵션은 최신 카탈로그·가격표·웹 매뉴얼에서 계속 확인하도록 문구를 유지했습니다.

- 현대 팰리세이드 하이브리드

## 정보 확인 필요 차량

- 더 뉴 그랜저 하이브리드 — 현재 판매 모델이 TMED-2 및 스테이 모드를 적용했다는 공식 근거 재확인 필요
- 향후 현대자동차 TMED-2 하이브리드 모델
- 향후 기아 TMED-2 하이브리드 모델
- 향후 제네시스 하이브리드 모델

확인되지 않은 차량은 스테이 모드 `applies` 데이터에 포함하지 않았습니다.

1. TMED-2 하이브리드 시스템
2. 스테이 모드
3. 스테이 모드 사용 예약
4. 하이브리드 V2L
5. 하이브리드 스마트 회생제동
6. 하이브리드 회생제동 단계 조절
7. 하이브리드 에너지 흐름도
8. 하이브리드 배터리 충전 상태
9. 하이브리드 전용 주행 정보
10. 하이브리드 정차 공조

`스테이 모드 사용 예약`과 `하이브리드 V2L`은 적용 범위가 확정되지 않아 검증 미완료 상태이며 차량 매핑을 제한했습니다.

## 스테이 모드 설명 정책

- 정차 중 고전압 배터리로 공조와 멀티미디어 등 편의 기능 사용
- EV 유틸리티 모드와 유사하나 하이브리드 배터리와 시스템에 맞게 작동
- 목적지 도착 전 충전 유도/예약은 차종 및 소프트웨어별 제공 가능성으로 표기
- 최대 약 1시간은 특정 조건의 안내 기준이며 실제 시간을 보장하지 않는다고 명시
- 배터리 잔량, 전력 사용량, 외기 온도와 시스템 상태에 따른 제한 포함
- 밀폐 공간에서 엔진 자동 작동 가능성에 관한 안전 경고 포함

## 관련 기능 링크

스테이 모드에서 다음 기능으로 이동할 수 있습니다.

- 유틸리티 모드
- 캠핑 모드
- V2L
- 충전 예약
- Battery Conditioning
- 하이브리드 스마트 회생제동
- TMED-2 하이브리드 시스템
- 스테이 모드 사용 예약

존재하지 않던 하이브리드 시스템, 예약, 스마트 회생제동 등의 항목은 실제 기능 데이터로 생성했습니다.

## 공식 출처

- 현대자동차그룹 차세대 하이브리드 시스템 공식 뉴스룸

- 현대자동차 공식 카탈로그·가격표 다운로드 센터

공식 페이지의 문구는 복사하지 않고 사용자 중심 설명으로 재작성했습니다. 직접 확인이 제한된 세부 트림과 향후 차종은 추정 매핑하지 않았습니다.
