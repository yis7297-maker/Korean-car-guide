Korean CAR GUIDE 이용방법

차량 기능을 검색하고, 적용 차량과 연식 차이를 확인하는 기본 방법을 안내합니다.

기능 검색 방법

검색창에 기능명을 입력하면 전체 기능 데이터에서 즉시 검색됩니다. 예를 들어 “RSPA”, “디지털 키”, “빌트인 캠”, “회생”, “V2L”처럼 한글명, 영문명, 약어, 관련 용어를 입력할 수 있습니다. 검색어가 입력된 상태에서는 카테고리 필터보다 검색 결과가 우선 적용되므로, 다른 카테고리에 있는 기능도 함께 찾을 수 있습니다.

차량 선택 방법

브랜드, 차종, 연식을 선택하면 해당 차량과 연결된 기능 목록을 확인할 수 있습니다. 같은 차종이라도 내연기관, 하이브리드, 전기차, 고성능 모델, N Line 여부에 따라 기능 구성이 다를 수 있으므로 차량명을 선택한 뒤 연식과 파워트레인을 함께 확인하는 것이 좋습니다.

적용 차량 확인 방법

기능 팝업 또는 기능 페이지의 적용 차량 영역은 현재 데이터베이스에 연결된 차종을 보여줍니다. 적용 차량 목록에 표시되더라도 실제 차량에는 트림 또는 옵션 조건이 붙을 수 있습니다. 구매 또는 정비 판단에는 반드시 제조사 공식 가격표, 카탈로그, 오너스 매뉴얼을 함께 확인해야 합니다.

연식 확인 방법

자동차 기능은 연식 변경과 소프트웨어 업데이트에 따라 메뉴명이나 작동 조건이 달라질 수 있습니다. 차량등록증의 최초 등록일만으로는 정확한 모델연식을 알기 어려울 수 있으므로, 차량 판매명, 생산 시기, 인포테인먼트 버전, 공식 가격표의 모델연식을 함께 확인하세요.

기능명 검색 팁

검색 결과가 적으면 공식 명칭, 약어, 영문명을 바꿔 검색해 보세요. 예를 들어 원격 스마트 주차 보조는 RSPA, Remote Smart Parking Assist로도 검색할 수 있고, 빌트인 캠은 Built-in Cam, 주행 녹화, 주차 중 녹화 같은 관련 용어로도 찾을 수 있습니다.

공식 매뉴얼 확인 방법

기능 페이지의 공식 출처를 먼저 확인한 뒤, 제조사 홈페이지의 오너스 매뉴얼 또는 웹 매뉴얼에서 같은 기능명을 검색하세요. 메뉴 경로나 버튼명이 페이지와 다르면 차량 화면과 공식 매뉴얼을 우선 기준으로 삼아야 합니다.
