## 기능 수

- 전체 기능 수: 11개
- 기능만 존재하고 적용 차량이 없는 데이터: 0개

## 기능-차량 매핑 수

- 기능-차량 매핑 수: 152건
- 모든 기능은 `applies` 필드로 적용 브랜드, 적용 차종, 적용 연식, 적용 트림, 선택 옵션 여부를 저장합니다.

## 최신 세대 기능 수

- 최신 세대 기능 수: 11개

최신 세대 우선 반영:
- RSPA → RSPA 2
- Digital Key → Digital Key 2 → Digital Key 2 Touch
- Built-in Cam → Built-in Cam 2 → Built-in Cam 2 Plus
- SCC/HDA → NSCC / HDA 2

각 기능은 `generation` 필드에 기능군, 현재 세대, 이전 세대, 세대별 차이를 저장합니다.

## 플레오스 기능 수

- 플레오스 기능 수: 3개

플레오스 기능:
- 플레오스 AI 음성 비서
- 플레오스 개인화 기능
- 플레오스 OTA / 앱 연동

처리 기준:
- 더 뉴 그랜저, 즉 앱의 `그랜저` 차량에 매핑했습니다.
- 기존 ccNC, 커넥티드, 인포테인먼트 기능과 혼동되지 않도록 `플레오스` 카테고리로 분리했습니다.
- 공식 카탈로그/가격표 원문 세부 확인이 끝나지 않아 `검증 미완료` 상태로 표시합니다.

## 검증 완료 기능 수

- 검증 완료 기능 수: 8개

검증 완료 기능:
- 원격 스마트 주차 보조 2 / RSPA 2
- 디지털 키 2 Touch
- 빌트인 캠 2 Plus
- NSCC / HDA 2
- EV 충전 예약 / 배터리 컨디셔닝 / V2L
- N 전용 주행/성능 기능
- 전방/후측방/후방 교차 충돌방지 보조
- 공조·시트·도어·디스플레이 편의 기능

## 검증 미완료 기능 수

- 검증 미완료 기능 수: 3개

검증 미완료 기능:
- 플레오스 AI 음성 비서
- 플레오스 개인화 기능
- 플레오스 OTA / 앱 연동

미완료 사유:
- 공개 검색에서 더 뉴 그랜저 플레오스 세부 기능의 공식 카탈로그/가격표 원문 URL을 독립적으로 확인하지 못했습니다.
- 플랫폼에는 표시하되 확정 기능과 구분되도록 `검증 미완료` 상태와 공식 출처 재확인 링크를 표시했습니다.

## 출처 검증 정책

각 기능은 아래 순서의 검증 상태를 데이터에 저장합니다.

1. 공식 카탈로그
2. 공식 가격표
3. 웹 매뉴얼
4. 오너스 매뉴얼
5. 공식 홈페이지

앱 상세 모달의 `출처 검증 상태`에서 각 항목의 완료/미완료 여부를 확인할 수 있습니다.
