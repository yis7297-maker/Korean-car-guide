## 필터 및 호출 검수

- 검색창이 차량 선택보다 먼저 배치됨
- 브랜드 전체 선택 시 브랜드 제한 해제
- 차종 전체 선택 시 차량 제한 해제
- 연식 전체 선택 시 연식 제한 해제
- 브랜드 변경 시 차종 선택이 전체로 초기화됨
- 기능 카드 클릭 및 키보드 Enter/Space로 상세 팝업 호출
- 관련 기능 버튼 클릭 시 대응 기능 팝업으로 전환
- 존재하지 않는 관련 기능은 버튼 또는 텍스트로 노출하지 않음
- JavaScript 문법 검사 통과
