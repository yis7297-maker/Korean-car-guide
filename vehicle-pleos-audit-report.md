# 차량 중심 DB 및 더 뉴 그랜저 플레오스 반영 리포트

작성일: 2026-06-17

## 추가된 더 뉴 그랜저 플레오스 관련 기능 목록

- 플레오스 커넥트 / Pleos Connect
- Gleo AI 음성 어시스턴트
- 플레오스 차량 앱 스토어

처리 방식:
- 기존 ccNC, 인포테인먼트, Bluelink/커넥티드 서비스와 혼동되지 않도록 `플레오스` 카테고리와 `플레오스` 태그로 분리했습니다.
- 적용 차종은 `현대 더 뉴 그랜저`로 등록했습니다.
- 전용/확장 필드에는 `더 뉴 그랜저 우선 적용, 향후 타 차종 확장 가능성 별도 관리`로 구분했습니다.
- 단, 공개 검색에서 현대 공식 플레오스 세부 카탈로그/가격표 원문 URL을 독립 확인하지 못해 앱 상태는 `출처 재확인 필요`로 표시했습니다.

## 현대자동차 전체 모델 반영 여부

반영 완료: 현대 28개 모델

- 세단: 아반떼, 쏘나타, 더 뉴 그랜저
- SUV: 베뉴, 코나, 투싼, 싼타페, 팰리세이드, 캐스퍼
- MPV: 스타리아, 스타리아 라운지
- 전기차: 캐스퍼 일렉트릭, 아이오닉 5, 아이오닉 6, 아이오닉 9, 포터 II 일렉트릭
- 수소전기차: 넥쏘
- 상용차: 포터 II, 마이티, 파비스, 카운티, 일렉시티, 유니버스, 엑시언트
- 고성능/N/N Line: 아반떼 N, 아이오닉 5 N, 쏘나타 N Line, 코나 N Line

## 고성능 모델/N/N Line 반영 여부

반영 완료.

- 아반떼 N
- 아이오닉 5 N
- 쏘나타 N Line
- 코나 N Line

고성능 전용 기능은 `고성능/N 전용` 카테고리와 `전용 기능`, `적용 차종 제한` 배지로 표시합니다.

구현 완료.

- 현대자동차, 기아, 제네시스 브랜드 카드 추가

- 차종 선택 시 해당 차량 적용 기능을 카테고리별 카드 리스트로 표시
- 차량별 기능 카드 클릭 시 상세 모달 표시

- 기능-차종 매핑 수: 앱 데이터 기준 150건 이상

## 팝업/모달 UI 적용 여부

적용 완료.

- 모바일에서는 바텀시트 형태로 표시됩니다.
- 모달 표시 항목: 기능명, 아이콘, 적용 브랜드/차종/연식/트림, 요약, 사용 방법, 작동 조건, 미작동 가능 상황, 주의/경고, 관련 기능, 공식 출처, 마지막 업데이트일

## 아직 공식 출처 재확인이 필요한 기능 목록

- Gleo AI 음성 어시스턴트

## 공식 출처 정책

- 앱의 공식 출처 링크는 브랜드 공식 다운로드 센터, 카탈로그/가격표, 웹/오너스 매뉴얼, 공식 기능 설명 페이지로 제한했습니다.
- 플레오스 관련 기능은 공식 원문 세부 URL 재확인 전까지 `출처 재확인 필요` 상태로 표시해 확정 기능과 분리했습니다.
