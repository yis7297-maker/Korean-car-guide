## 차량 역검색

기능의 `applies` 데이터를 역방향으로 읽어 다음 정보를 제공합니다.

- 적용 브랜드와 차량
- 적용 연식
- 적용 트림 또는 패키지
- 기본/선택 사양 여부

연식이나 트림이 아직 세분화되지 않은 데이터는 추정하지 않고 기존 공식 자료 매핑 문구를 그대로 표시합니다.

## EV 전용 기능

신규 추가 기능:

- 충전 상태 확인
- 에너지 사용량
- 배터리 상태 확인
- i-Pedal
- EV 스마트 회생제동
- 회생제동 단계 조절

기존 전동화 기능을 `EV 전용 기능` 카테고리로 통합하고 다음 하위 분류를 적용했습니다.

- 충전
- 배터리
- 회생제동
- 전력 공급
- 에너지 관리
- EV 편의 기능

V2H와 V2G는 `향후 지원 예정` 태그를 유지합니다.
