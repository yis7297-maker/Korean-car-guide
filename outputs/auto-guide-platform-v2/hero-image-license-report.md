## 라이선스 확인 결과

공식 뉴스룸과 PR 페이지의 이미지는 보도·홍보 목적 제공 자료일 수 있으나, 제3자 운영 검색 서비스의 상업적 웹 배너로 재사용할 수 있다는 명시적 허가 조건을 이번 작업 환경에서 확인하지 못했습니다. 공식 자료라는 사실만으로 자유로운 상업적 재사용 권한이 생기지 않으므로 적용 대상에서 제외했습니다.

- 현대자동차 공식 PR 후보: 상업 웹사이트 재사용 허가 확인 불가 — 미적용
- 기아 공식 PR 후보: 상업 웹사이트 재사용 허가 확인 불가 — 미적용
- 제네시스 공식 PR 후보: 상업 웹사이트 재사용 허가 확인 불가 — 미적용
- 출처가 불명확한 검색 이미지·재배포 이미지: 검토 제외 및 사용 금지

실제 공식 PR 이미지를 사용하려면 각 브랜드의 서면 사용 허가 또는 해당 이미지 페이지에 명시된 재사용 라이선스 조건을 별도로 확보해야 합니다.

## 최종 적용 이미지

- 파일: `assets/hero-original-mobility-v2.png`
- 형식: 고해상도 가로형 PNG
- 제작 방식: 이미지 생성 도구로 새로 제작한 브랜드 중립 자동차 비주얼
- 제3자 입력 이미지: 사용하지 않음
- 브랜드 로고·차명·상표: 사용하지 않음
- 실제 양산차 디자인 복제: 배제하도록 생성 지시
- 워터마크·텍스트: 없음

생성형 이미지의 최종 상업 이용은 운영 주체가 사용하는 생성 서비스 약관과 관할 법률을 함께 확인해야 합니다. 본 이미지에는 확인되지 않은 외부 사진을 합성하지 않았습니다.
