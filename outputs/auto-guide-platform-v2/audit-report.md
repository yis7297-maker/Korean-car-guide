## 수정 완료된 기능 목록

- 기존 `공식 확인 완료`/`공식 출처 확인 필요` 이원 구조를 `확정` 중심 구조로 정리했습니다.
- 기능 상세 페이지에 필수 항목 10개를 모두 표시하도록 수정했습니다.
- 현대/기아/제네시스 공통 기능은 중복 페이지 대신 한 페이지에서 브랜드별 차이를 보여주도록 수정했습니다.
- 안전 관련 기능은 운전자 책임, 센서 한계, 악천후·센서 오염·차선 인식 실패 등 제한 조건을 보수적으로 재작성했습니다.
- 검색 결과 카드에서 브랜드와 확정 상태, 적용 범위를 함께 보여주도록 수정했습니다.

## 저작권/표현 검수

- 매뉴얼 문장을 그대로 저장하거나 노출하지 않고, 검색 서비스용 문장으로 새로 작성했습니다.
- ADAS, 주차 보조, 원격 제어, 전동화 기능은 센서 한계와 운전자 책임을 축소하지 않았습니다.
- 동일 기능의 브랜드별 명칭 차이는 관련 기능과 검색 키워드로 연결했습니다.
