EV 전용 기능

스마트 회생제동

전방 차량과 주행 상황을 고려해 회생 제동량을 자동으로 조절하는 기능입니다.

공식 검증 완료

기능 개요

전방 차량과 주행 상황을 고려해 회생 제동량을 자동으로 조절하는 기능입니다.

사용 전 조건

  • 스마트 회생 제동 기능이 지원되는 전동화 차량이어야 합니다.
  • 전방 센서가 정상 상태여야 합니다.

설정 방법

  1. EV 또는 주행 보조 관련 메뉴에서 스마트 회생 제동 기능을 설정합니다.

사용 방법

  1. 스마트 회생 제동 기능을 켭니다.
  2. 주행 중 전방 차량과 도로 상황에 따라 회생 제동량이 자동 조절되는지 확인합니다.
  3. 필요 시 패들 시프트로 회생 제동 단계를 조절합니다.

해제 방법

  1. 스마트 회생 제동 설정을 해제하거나 패들 시프트 조작으로 수동 회생 제동 단계로 전환합니다.

작동 제한 상황

  • 전방 센서 오염, 악천후, 급커브, 전방 차량 인식 불량 시 작동이 제한될 수 있습니다.

주의 사항

  • 회생 제동 기능에만 의존하지 말고 필요한 경우 브레이크 페달을 직접 조작하십시오.
