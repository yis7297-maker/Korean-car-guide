기능 개요

하이패스 시스템 / e-Hi Pass 기능의 개요와 적용 차량 정보를 제공합니다. 세부 조작 절차는 공식 매뉴얼에서 확인된 항목만 표시합니다.

사용 전 조건

  • 하이패스 카드가 올바르게 삽입되어 있어야 합니다.
  • 차량 전원이 켜진 상태에서 하이패스 단말기 표시등과 음성 안내를 확인합니다.

설정 방법

  1. 하이패스 시스템 장착 위치와 카드 삽입 방향을 확인합니다.
  2. 필요 시 하이패스 카드 잔액, 등록 상태, 단말기 동작 상태를 점검합니다.

사용 방법

  1. 하이패스 카드를 단말기에 삽입합니다.
  2. 표시등 또는 안내음으로 동작 상태를 확인합니다.
  3. 하이패스 차로를 규정 속도에 맞춰 통과합니다.

해제 방법

  1. 하이패스 카드를 분리하거나 하이패스 미사용 차로를 이용합니다.

작동 제한 상황

  • 카드 미삽입, 카드 오류, 단말기 이상, 통신 상태 불량 시 정상 결제가 되지 않을 수 있습니다.

주의 사항

  • 하이패스 차로에서는 안내 속도를 준수하고, 결제 오류가 발생해도 급정지하거나 후진하지 마십시오.
