편의 기능

원격 시동

원격 시동은 스마트 키 또는 커넥티드 서비스 앱을 통해 차량 밖에서 시동 또는 공조를 미리 작동하는 기능입니다.

공식 검증 완료

기능 개요

원격 시동은 스마트 키 또는 커넥티드 서비스 앱을 통해 차량 밖에서 시동 또는 공조를 미리 작동하는 기능입니다.

사용 전 조건

  • 스마트 키 또는 커넥티드 서비스 원격 제어 조건을 만족해야 합니다.
  • 차량이 잠금 상태이고 변속 위치 등 안전 조건을 만족해야 합니다.

설정 방법

  1. 스마트 키 버튼 또는 커넥티드 서비스 앱의 원격 시동/공조 메뉴를 사용합니다.

사용 방법

  1. 차량을 잠근 뒤 원격 시동 버튼 또는 앱 원격 시동을 실행합니다.
  2. 차량의 비상등 또는 앱 상태 표시로 작동 여부를 확인합니다.

해제 방법

  1. 원격 시동 취소 버튼을 누르거나 앱에서 원격 제어를 종료합니다.
  2. 정해진 시간이 지나면 원격 시동이 자동 종료될 수 있습니다.

작동 제한 상황

  • 도어/후드/테일게이트 열림, 스마트 키 위치, 배터리 상태, 통신 상태에 따라 작동하지 않을 수 있습니다.

주의 사항

  • 밀폐된 장소에서 원격 시동을 사용하지 마십시오.
