EV 전용 기능

충전 예약

사용자가 설정한 시간대에 충전이 시작되도록 예약하는 전동화 차량 기능입니다.

공식 검증 완료

기능 개요

사용자가 설정한 시간대에 충전이 시작되도록 예약하는 전동화 차량 기능입니다.

사용 전 조건

  • 충전 케이블이 차량에 연결되어 있어야 합니다.
  • 예약 충전 기능이 지원되는 전동화 차량이어야 합니다.

설정 방법

  1. EV 메뉴 또는 충전 관리 화면에서 예약 충전 시간을 설정합니다.
  2. 필요 시 출발 시간 또는 저렴한 전기 시간대 조건을 함께 설정합니다.

사용 방법

  1. 충전 커넥터를 차량에 연결합니다.
  2. 예약 충전 시간을 설정합니다.
  3. 차량 화면에 예약 충전 설정 메시지가 표시되는지 확인합니다.
  4. 예약 시간이 되면 충전이 시작됩니다.

해제 방법

  1. 충전 예약 메뉴에서 예약 충전을 해제합니다.
  2. 즉시 충전이 필요한 경우 예약 충전 대기 상태를 해제하고 충전을 시작합니다.

작동 제한 상황

  • 예약 공조, 충전기 상태, 배터리 잔량, 외부 전원 상태에 따라 충전 예상 시간이 달라질 수 있습니다.

주의 사항

  • 예약 충전 중에도 충전 커넥터와 충전기 상태를 확인하십시오.
