주차 보조

주차 거리 경고

저속 주차 중 주변 장애물과의 거리를 경고음과 화면으로 알려주는 기능입니다.

공식 검증 완료

기능 개요

저속 주차 중 주변 장애물과의 거리를 경고음과 화면으로 알려주는 기능입니다.

사용 전 조건

  • 차량 전원이 켜져 있고 주차 보조 기능이 지원되는 차량이어야 합니다.
  • 카메라와 초음파 센서가 가려져 있지 않아야 합니다.

설정 방법

  1. 설정 > 차량 > 운전자 보조 > 주차 안전 또는 주차 편의 메뉴에서 해당 기능을 선택합니다.

사용 방법

  1. 저속 주행 또는 변속 위치 조건에서 주차 보조 화면이나 경고 표시를 확인합니다.
  2. 주차/뷰 버튼 또는 변속 위치 조작으로 화면과 경고 상태를 확인합니다.
  3. 주변을 직접 확인하면서 주차 또는 후진 조작을 진행합니다.

해제 방법

  1. 주차/뷰 버튼을 다시 누르거나 변속 위치를 변경합니다.
  2. 설정 메뉴에서 해당 주차 보조 기능을 해제합니다.

작동 제한 상황

  • 센서 오염, 낮은 장애물, 얇은 물체, 경사로, 비/눈/결빙 상태에서는 감지 성능이 제한될 수 있습니다.

주의 사항

  • 주차 보조 표시와 경고에만 의존하지 말고 운전자가 직접 주변을 확인하십시오.
