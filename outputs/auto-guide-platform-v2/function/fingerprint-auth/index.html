지문 인증

등록된 지문으로 사용자 프로필, 시동 또는 일부 편의 기능 인증을 수행하는 기능입니다.

공식 검증 완료

기능 개요

등록된 지문으로 사용자 프로필, 시동 또는 일부 편의 기능 인증을 수행하는 기능입니다.

사용 전 조건

  • 지문 인증 시스템이 지원되는 차량이어야 합니다.
  • 사용자 프로필이 설정되어 있어야 합니다.
  • 손가락과 지문 센서 표면이 깨끗해야 합니다.

설정 방법

  1. 설정 > 사용자 프로필 > 보안 > 지문 인증 메뉴로 진입합니다.
  2. 지문 등록 또는 삭제 메뉴를 선택합니다.

사용 방법

  1. 등록할 사용자 프로필을 선택합니다.
  2. 화면 안내에 따라 지문 센서에 손가락을 올립니다.
  3. 여러 방향으로 지문을 인식시켜 등록을 완료합니다.
  4. 인증이 필요한 기능에서 등록한 지문을 센서에 접촉합니다.

해제 방법

  1. 지문 인증 메뉴에서 등록된 지문을 삭제합니다.
  2. 필요 시 사용자 프로필의 보안 설정에서 지문 사용을 해제합니다.

작동 제한 상황

  • 손가락이 젖었거나 센서가 오염된 경우 인식이 제한될 수 있습니다.
  • 상처, 장갑, 지문 상태에 따라 인증이 실패할 수 있습니다.

주의 사항

  • 타인의 지문을 등록하지 마십시오.
  • 보안 기능 해제 또는 지문 삭제 전 대체 인증 방법을 확인하십시오.
