스마트 파워 테일게이트

스마트 파워 테일게이트는 스마트 키를 소지한 상태에서 테일게이트 주변에 머물거나 스위치를 조작해 테일게이트를 자동으로 열고 닫는 기능입니다.

공식 검증 완료

기능 개요

스마트 파워 테일게이트는 스마트 키를 소지한 상태에서 테일게이트 주변에 머물거나 스위치를 조작해 테일게이트를 자동으로 열고 닫는 기능입니다.

사용 전 조건

  • 스마트 키를 휴대하고 차량 후방 감지 영역에 접근합니다.
  • 차량 도어 잠금 상태와 테일게이트 작동 가능 조건을 확인합니다.

설정 방법

  1. 인포테인먼트 시스템 또는 차량 설정 메뉴에서 도어/테일게이트 관련 설정을 확인합니다.
  2. 스마트 열림 또는 파워 테일게이트 기능을 켭니다.

사용 방법

  1. 스마트 키를 휴대한 상태로 테일게이트 뒤쪽에 접근합니다.
  2. 안내음 또는 비상등 점멸 후 테일게이트가 자동으로 열리는지 확인합니다.
  3. 테일게이트 닫힘 버튼 또는 스마트 키/실내 스위치를 눌러 닫습니다.

해제 방법

  1. 차량 설정에서 스마트 테일게이트 또는 파워 테일게이트 기능을 끕니다.
  2. 작동 중 테일게이트 스위치를 누르면 작동이 정지될 수 있습니다.

작동 제한 상황

  • 스마트 키 위치, 주변 전파 환경, 장애물, 경사 상태에 따라 작동하지 않을 수 있습니다.
  • 테일게이트 주변에 장애물이 있으면 자동 작동이 제한될 수 있습니다.

주의 사항

  • 테일게이트 작동 전 주변 사람이나 물체가 없는지 확인하십시오.
  • 닫힘 중 신체나 물건이 끼이지 않도록 주의하십시오.
