스마트 파워 테일게이트

파워 테일게이트 전동 개폐와 스마트 키 감지 기반 자동 열림을 제공하는 편의 기능입니다.

기능 개요

  1. 파워 테일게이트 전동 개폐와 스마트 키 감지 기반 자동 열림을 제공하는 편의 기능입니다.

사용 전 조건

  1. 파워 테일게이트 또는 스마트 테일게이트 사양이 적용된 차량이어야 합니다.
  2. 파워 테일게이트는 변속 위치 P(주차) 또는 시동 OFF 조건에서 작동합니다.
  3. 스마트 테일게이트는 모든 도어가 닫히고 잠긴 뒤 일정 시간이 지나야 작동합니다.
  4. 테일게이트 주변에 충분한 공간이 있어야 합니다.

설정 방법

  1. 인포테인먼트 시스템에서 설정 > 차량 > 도어/테일게이트로 이동합니다.
  2. 파워 테일게이트 열림 속도를 보통 또는 빠르게 선택합니다.
  3. 파워 테일게이트 열림 높이를 완전열림, 단계별 높이 또는 사용자 높이 설정으로 선택합니다.
  4. 스마트 테일게이트 사용 여부를 설정합니다.

사용 방법

  1. 스마트 키의 테일게이트 버튼을 길게 누르거나 운전석 테일게이트 버튼을 눌러 테일게이트를 엽니다.
  2. 번호판 상단의 외부 버튼을 누르면 경고음과 함께 테일게이트가 열립니다.
  3. 테일게이트 내부 열림/닫힘 버튼을 누르면 테일게이트가 열리거나 닫힙니다.
  4. 스마트 테일게이트가 설정된 경우 스마트 키를 휴대하고 뒤쪽 감지 영역에 머물면 경보 후 테일게이트가 자동으로 열립니다.

해제 방법

  1. 스마트 테일게이트 감지·경보 중 도어 잠금, 도어 잠금 해제, 테일게이트 작동 버튼을 짧게 누르면 기능이 중지됩니다.
  2. 인포테인먼트 시스템의 도어/테일게이트 설정에서 스마트 테일게이트를 해제합니다.
  3. 파워 테일게이트 작동 중 버튼을 다시 누르면 작동이 멈추거나 반전될 수 있습니다.

작동 제한 상황

  1. 차량이 3 km/h 이상으로 이동 중이면 파워 테일게이트를 열 수 없습니다.
  2. 반복 작동 시 과열 방지 모드로 진입해 일정 시간 작동하지 않을 수 있습니다.
  3. 버튼 주변이 얼었거나 차량이 기울어진 상태, 경사지, 전파 혼선 지역에서는 작동이 지연되거나 제한될 수 있습니다.

주의 사항

  1. 테일게이트 작동 전 주변에 사람이나 물체가 없는지 확인하십시오.
  2. 차 안에 어린이나 애완동물만 남겨 둔 상태로 차량을 떠나지 마십시오.
  3. 물체 끼임 인식 확인을 위해 신체나 물건을 작동 범위에 두지 마십시오.
  4. 세차 시 스마트 테일게이트 기능을 중지하십시오.
