기능 개요

원격 스마트 출차 보조 기능의 개요와 적용 차량 정보를 제공합니다. 세부 조작 절차는 공식 매뉴얼에서 확인된 항목만 표시합니다.

사용 전 조건

  • 스마트 키를 소지하고 차량 주변이 안전해야 합니다.
  • 기능 지원 차량이며 원격 조작 조건을 만족해야 합니다.

설정 방법

  1. 운전자 보조 또는 주차 안전 관련 메뉴에서 원격 스마트 주차 보조 설정을 확인합니다.

사용 방법

  1. 차량 밖에서 스마트 키를 이용해 차량을 원격 시동 상태로 준비합니다.
  2. 전진 또는 후진 버튼을 길게 눌러 차량을 이동시킵니다.
  3. 차량 주변을 계속 확인하며 버튼을 놓으면 차량 이동이 멈춥니다.

해제 방법

  1. 스마트 키 버튼에서 손을 떼거나 원격 시동을 해제합니다.

작동 제한 상황

  • 경사로, 장애물, 좁은 공간, 노면 상태, 센서 오염 상태에서는 작동이 제한될 수 있습니다.

주의 사항

  • 운전자는 차량 이동 중 주변 안전을 직접 확인해야 하며, 사람이나 물체가 있으면 즉시 조작을 중지하십시오.
