메모리 시트

메모리 시트는 운전석 위치와 관련 장치 위치를 저장하고 불러오는 기능입니다.

공식 검증 완료

기능 개요

메모리 시트는 운전석 위치와 관련 장치 위치를 저장하고 불러오는 기능입니다.

사용 전 조건

  • 차량을 안전한 곳에 정차하고 P단 또는 정차 상태를 유지합니다.

설정 방법

  1. 운전석, 스티어링 휠, 실외 미러, HUD 위치를 운전자에 맞게 조절합니다.

사용 방법

  1. 저장 버튼 또는 1/2 메모리 버튼을 길게 눌러 현재 운전 자세를 저장합니다.
  2. 저장된 번호 버튼을 눌러 운전 자세를 불러옵니다.

해제 방법

  1. 작동 중 좌석 조절 스위치나 메모리 버튼을 조작하면 이동이 중지될 수 있습니다.

작동 제한 상황

  • 변속 상태, 차량 속도, 도어 상태에 따라 불러오기 작동이 제한될 수 있습니다.

주의 사항

  • 좌석 이동 중 탑승자나 물건이 끼이지 않도록 주의하십시오.
