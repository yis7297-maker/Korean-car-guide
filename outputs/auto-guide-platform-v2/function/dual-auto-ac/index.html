듀얼 풀오토 에어컨

듀얼 풀오토 에어컨은 운전석과 동승석의 실내 온도, 풍량, 풍향을 자동 또는 개별로 조절하는 기능입니다.

공식 검증 완료

기능 개요

듀얼 풀오토 에어컨은 운전석과 동승석의 실내 온도, 풍량, 풍향을 자동 또는 개별로 조절하는 기능입니다.

사용 전 조건

  • 시동 또는 전원 상태에서 공조 장치를 사용할 수 있어야 합니다.

설정 방법

  1. 공조 조작부 또는 인포테인먼트 공조 화면에서 온도, 풍량, 풍향, AUTO를 설정합니다.

사용 방법

  1. AUTO 버튼을 눌러 자동 공조를 켭니다.
  2. 운전석/동승석 온도를 원하는 값으로 조절합니다.
  3. 필요하면 SYNC, 내외기, 앞유리 김서림 제거, A/C를 조절합니다.

해제 방법

  1. OFF 버튼을 누르거나 풍량을 끄면 공조 작동이 중지됩니다.

작동 제한 상황

  • 실내외 온도, 습도, 유리 김서림 상태에 따라 자동 제어 방식이 달라질 수 있습니다.

주의 사항

  • 장시간 내기 순환 사용은 실내 공기질 저하나 유리 김서림을 유발할 수 있습니다.
