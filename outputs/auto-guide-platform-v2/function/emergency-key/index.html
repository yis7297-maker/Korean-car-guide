비상 키

비상 키는 스마트 키 배터리 방전 등 비상 상황에서 도어 잠금/해제에 사용하는 기계식 키입니다.

공식 검증 완료

기능 개요

비상 키는 스마트 키 배터리 방전 등 비상 상황에서 도어 잠금/해제에 사용하는 기계식 키입니다.

사용 전 조건

  • 스마트 키 내부의 비상 키를 분리할 수 있어야 합니다.

설정 방법

  1. 별도 설정 없이 스마트 키에서 비상 키를 분리해 사용합니다.

사용 방법

  1. 스마트 키의 분리 버튼을 눌러 비상 키를 꺼냅니다.
  2. 도어 키 홀에 비상 키를 넣고 돌려 도어를 잠금 또는 해제합니다.

해제 방법

  1. 사용 후 비상 키를 스마트 키에 다시 장착합니다.

작동 제한 상황

  • 비상 키는 도어 잠금/해제 등 제한된 기능에 사용됩니다.

주의 사항

  • 비상 키 사용 후 스마트 키 배터리 상태를 점검하십시오.
