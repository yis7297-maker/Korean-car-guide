편의 기능

헤드업 디스플레이 / HUD

헤드업 디스플레이는 주행 정보 일부를 운전자 전방 시야 영역에 표시하는 기능입니다.

공식 검증 완료

기능 개요

헤드업 디스플레이는 주행 정보 일부를 운전자 전방 시야 영역에 표시하는 기능입니다.

사용 전 조건

  • HUD 적용 차량에서 전원 또는 시동 상태가 켜져 있어야 합니다.

설정 방법

  1. 설정 메뉴에서 헤드업 디스플레이 표시, 높이, 회전, 밝기, 표시 정보를 조절합니다.

사용 방법

  1. HUD 표시를 켭니다.
  2. 운전자 시야에 맞춰 높이와 밝기를 조절합니다.
  3. 표시할 주행 정보 항목을 선택합니다.

해제 방법

  1. 설정 메뉴에서 헤드업 디스플레이 표시를 끕니다.

작동 제한 상황

  • 운전자 착좌 위치, 선글라스, 전면 유리 상태, 밝은 외부 환경에 따라 시인성이 달라질 수 있습니다.

주의 사항

  • HUD 조정은 안전한 상태에서 수행하고, 주행 중 화면 조작에 주의하십시오.
