EV 전용 기능

EV 충전 예약 / 배터리 컨디셔닝 / V2L

고전압 배터리 전력을 차량 외부 또는 실내 전기 장치에 공급하는 기능입니다.

기능 개요

고전압 배터리 전력을 차량 외부 또는 실내 전기 장치에 공급하는 기능입니다.

사용 전 조건

  • 전기 사용(V2L) 기능이 지원되는 차량이어야 합니다.
  • 고전압 배터리 잔량이 설정된 방전 제한량보다 높아야 합니다.
  • V2L 커넥터 또는 실내 전원 장치가 정상 연결되어야 합니다.

설정 방법

  1. EV 메뉴에서 전기 사용(V2L) 방전 제한량을 설정합니다.
  2. 필요 시 실내 또는 실외 V2L 사용 조건을 차량 화면에서 확인합니다.

사용 방법

  1. V2L 커넥터 또는 실내 전원 장치에 전기 제품을 연결합니다.
  2. 차량 화면 또는 커넥터 상태 표시를 확인합니다.
  3. 전기 제품 사용 중 배터리 잔량과 사용 가능 시간을 확인합니다.

해제 방법

  1. 전기 제품 사용을 중지하고 플러그를 분리합니다.
  2. 실외 V2L 사용 시 커넥터 잠금 해제 후 커넥터를 분리합니다.

작동 제한 상황

  • 고전압 배터리 잔량, 사용 전력, 외부 온도, 커넥터 상태에 따라 사용 시간이 제한될 수 있습니다.

주의 사항

  • 정격 용량을 초과하는 전기 제품을 연결하지 마십시오.
  • 젖은 손이나 습한 환경에서 커넥터를 조작하지 마십시오.
