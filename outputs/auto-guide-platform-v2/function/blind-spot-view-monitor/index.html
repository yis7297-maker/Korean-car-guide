후측방 모니터 / BVM

방향지시등 작동 시 해당 방향의 후측방 영상을 클러스터에 표시합니다.

기능 개요

  1. 방향지시등 작동 시 해당 방향의 후측방 영상을 클러스터에 표시합니다.

사용 전 조건

  1. 후측방 모니터가 차량 설정에서 사용하도록 설정되어 있어야 합니다.
  2. 사이드 미러 카메라가 가려져 있지 않아야 합니다.

설정 방법

  1. 설정 → 차량 → 운전자 보조 → 주행 안전 → 후측방 모니터

사용 방법

  1. 좌측 또는 우측 방향지시등을 작동합니다.
  2. 클러스터에 표시된 해당 방향의 후측방 영상을 확인합니다.

해제 방법

  1. 방향지시등을 끄면 영상이 종료됩니다.
  2. 차량 설정에서 후측방 모니터를 끌 수 있습니다.

작동 제한 상황

  1. 카메라 렌즈 오염, 악천후와 조도 조건에 따라 영상 확인이 제한될 수 있습니다.

주의 사항

  1. 카메라 영상과 경고는 주변 확인을 대신하지 않습니다. 운전자가 직접 차량 주변을 확인하고 제동과 조향을 수행해야 합니다.

관련 기능

서라운드 뷰 모니터 / SVM

적용 차량 보기

현재 구조화된 적용 차량 데이터가 없습니다.

공식 출처

  1. 현대자동차 공식 웹/오너스 매뉴얼 3순위/4순위
  2. 기아 공식 사용설명서 3순위/4순위
  3. 제네시스 공식 다운로드 센터 1순위-4순위
