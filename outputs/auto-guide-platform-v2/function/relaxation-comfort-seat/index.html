기능 개요

릴렉션 컴포트 시트 기능의 개요와 적용 차량 정보를 제공합니다. 세부 조작 절차는 공식 매뉴얼에서 확인된 항목만 표시합니다.

사용 전 조건

  • 차량이 안전하게 정차해 있고 릴렉션 컴포트 시트가 장착되어 있어야 합니다.

설정 방법

  1. 시트 조절 스위치 또는 시트 메뉴에서 릴렉션 기능을 선택합니다.

사용 방법

  1. 릴렉션 컴포트 시트 버튼 또는 메뉴를 선택합니다.
  2. 시트가 릴렉션 위치로 이동하는 동안 주변 공간을 확인합니다.
  3. 필요 시 수동 조절 스위치로 위치를 보정합니다.

해제 방법

  1. 시트 조절 스위치 또는 릴렉션 버튼을 조작해 일반 착좌 위치로 복귀합니다.

작동 제한 상황

  • 2열 탑승자, 적재물, 시트 주변 간섭이 있으면 작동이 제한될 수 있습니다.

주의 사항

  • 주행 중에는 릴렉션 자세를 사용하지 말고 안전벨트를 올바르게 착용하십시오.
