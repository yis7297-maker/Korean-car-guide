# Korean CAR GUIDE 최종 검수

- Pleos 기능 재검증 결과: 기존 4개 항목은 모두 공식 적용 트림·기능 범위가 확정되지 않은 후보 데이터였습니다. 추정 정보 노출 금지 원칙에 따라 전부 제거했으며, 공식 자료로 확인되는 기능만 추후 재등록하도록 처리했습니다.
- 신규 추가 기능 목록: 메모리 리버스 어시스트(MRA)
- MRA 추가 여부: 완료. 저장 방법, 활성화 방법, 후진 경로 재생 방법, 제한 조건 포함
- 차세대 FMC 차량 추가 여부: 완료. 아반떼(CN8 후속), 투싼(NX5 후속) — 기능 데이터 없이 정보 제공 예정 문구만 표시
- 관련 기능 하이퍼링크 적용 여부: 완료. 등록된 관련 기능은 상세 팝업으로 이동
- 픽토그램 적용 기능 수: 27개(현재 공개 기능 전체)
- 지원 상태 탭 삭제 여부: 완료
- 방문자 통계 위젯 적용 여부: 완료. 브라우저 로컬 집계 방식이며 푸터 버튼으로 ON/OFF 가능
- Footer 디스클레이머 적용 여부: 완료. 출처 검증 관련 문구는 푸터에서만 노출

## Pleos 검증 메모

우선 확인 대상으로 지정된 더 뉴 그랜저 공식 카탈로그, 가격표, 현대자동차 공식 홈페이지, 웹 매뉴얼을 기준으로 기존 데이터의 확정 근거 상태를 재점검했습니다. 기존 앱에 등록된 Pleos Connect, AI 음성 비서, 개인화, OTA/앱 연동 항목은 모두 자체적으로도 `재확인 필요`, `후보 기능`, `검증 미완료`로 기록되어 있었습니다. 따라서 추정 내용을 확정 정보처럼 보여주지 않도록 공개 데이터에서 제거했습니다.
