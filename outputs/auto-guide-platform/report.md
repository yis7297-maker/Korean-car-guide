## 구현 완료 목록

- 데이터 상태를 `공식 확인 완료`와 `공식 출처 확인 필요`로 분리했습니다.
- 기능 상세에 기능명, 브랜드/차종 적용 범위, 기능 요약, 사용 방법, 작동 조건, 작동하지 않을 수 있는 상황, 주의/경고 사항, 관련 기능, 공식 출처 링크, 마지막 업데이트일을 표시하도록 바꿨습니다.
- 공식 출처가 확인되지 않은 기능은 확정 설명처럼 보이지 않도록 상세 사용법을 숨기고 후보 상태로만 노출했습니다.
- 검색 키워드에 공식명, 영문 약칭, 일상 표현을 함께 넣었습니다.
- 같은 기능이 여러 브랜드에 걸치는 경우 공통 설명과 브랜드별 출처를 한 상세 화면에서 보여주도록 구조화했습니다.
- 검색 결과는 검색어 관련도와 공식 확인 상태를 함께 반영해 정렬합니다.

## 누락된 기능 목록

아래 범주는 아직 세부 기능 데이터로 쪼개지지 않았습니다. 공식 출처 확인 후 별도 기능으로 확장해야 합니다.

- 내비게이션 목적지 전송, 경유지, 실시간 교통, OTA 업데이트
- 운전자 주의 경고, 전방 주시 경고, 후석 승객 알림
- 지능형 속도 제한 보조, 내비게이션 기반 스마트 크루즈 컨트롤
- 원격 창문/선루프 제어, 워크 어웨이 잠금, 전자식 차일드락
- 열선/통풍/스티어링 휠 열선, 후석 공조, 공기청정 모드
- 전자식 변속 버튼/다이얼, 오토홀드, 전자식 파킹 브레이크
- AWD/터레인 모드, 프리뷰 전자제어 서스펜션, 후륜 조향
- EV 주행 가능 거리 표시, 충전 한도, 충전 커넥터 잠금, i-PEDAL
- V2L, V2G/V2H 적용 기능, 배터리 프리컨디셔닝 세부 조건
- 빌트인 캠, 디지털 센터 미러, 후방 모니터 세부 설정
- 무선 소프트웨어 업데이트, 사용자 프로필 백업/복원
- 지문 인증, 얼굴 인증 등 생체 인증 세부 기능
