주행 편의

NSCC / HDA 2

고속도로에서 속도, 차간 거리, 차로 중앙 주행을 보조합니다.

기능 개요

  • 고속도로에서 속도, 차간 거리, 차로 중앙 주행을 보조합니다.

사용 전 조건

  • 차량이 D(주행) 상태여야 합니다.
  • 차선과 전방 차량이 인식되어야 합니다.
  • 스마트 크루즈와 차로 보조가 켜져 있어야 합니다.

사용 방법

  • 스티어링 휠의 주행 보조/크루즈 버튼을 누릅니다.
  • SET 스위치로 원하는 속도를 설정합니다.
  • 차간 거리 버튼으로 앞차와의 간격을 선택합니다.
  • 클러스터에 차선, 앞차, HDA/HDA 2 표시가 켜졌는지 확인합니다.
  • 손은 스티어링 휠에 둔 상태로 주행합니다.

해제 방법

  • 브레이크 페달을 밟거나 CANCEL 버튼을 누릅니다.
  • 주행 보조 버튼을 다시 눌러 기능을 끕니다.

작동 제한 상황

  • 공사 구간, 급커브, 터널, 차선 훼손, 악천후, 센서 오염 시 제한됩니다.

주의 사항

  • 자율주행 기능이 아니며 즉시 조향/제동할 준비가 필요합니다.

관련 기능

  • 전방 충돌방지 보조
