하이브리드 배터리 충전 상태

고전압 배터리 충전 상태를 클러스터에서 확인하는 하이브리드 전용 표시 기능입니다.

기능 개요

  • 스타리아 하이브리드 매뉴얼 기준 고전압 배터리 충전량 표시계는 시동 ON 상태에서 고전압 배터리 충전 상태를 표시합니다.

사용 전 조건

  • 시동 ON 상태여야 합니다.

설정 방법

  • 클러스터에서 고전압 배터리 충전량 표시계를 확인합니다.

사용 방법

  • 클러스터의 고전압 배터리 충전량 표시계를 확인합니다.
  • L 위치는 용량 부족, H 위치는 완전 충전 상태를 의미합니다.

해제 방법

  • 표시 항목을 다른 클러스터 화면으로 변경합니다.

작동 제한 상황

  • 표시 상태는 주행 및 충전 상태에 따라 변동됩니다.

주의 사항

  • 배터리 관련 경고가 표시되면 안전한 곳에 정차 후 안내에 따라 조치하십시오.

관련 기능

  • 하이브리드 에너지 흐름도
