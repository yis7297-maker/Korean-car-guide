후방 모니터

후방 모니터 / RVM

광각 후방 카메라 영상으로 차량 후방 상황을 모니터에 표시해 주차 또는 저속 주행을 돕는 기능입니다.

공식 검증 완료

기능 설명

광각 후방 카메라 영상으로 차량 후방 상황을 모니터에 표시해 주차 또는 저속 주행을 돕는 기능입니다.

사용 전 조건

  • 광각 후방 카메라가 적용된 차량이어야 합니다.
  • 시동 'ON' 상태에서 관련 표시 정보를 설정할 수 있습니다.

설정 방법

  1. 설정 > 차량 > 운전자 보조 > 경고 방식 > 주차 안전 우선
  2. 후방 모니터 화면의 설정 버튼 또는 설정 > 차량 > 운전자 보조 > 주차 안전 > 카메라 설정
  3. 표시 정보에서 후방 모니터 유지와 후방 주차 가이드라인을 설정합니다.

사용 방법

  1. 'R'(후진)로 변속하면 후방 영상이 표시됩니다.
  2. 주차/뷰 버튼으로 후방 뷰 기능을 켜거나 끌 수 있습니다.
  3. 후방 모니터 화면의 뷰 전환 버튼으로 후방 뷰, 후방 탑 뷰, 후방 와이드 뷰를 선택할 수 있습니다.
  4. 후방 모니터 유지가 설정되어 있으면 'R'에서 'N' 또는 'D'로 변속해도 조건에 따라 후방 영상이 유지됩니다.

해제 방법

  1. 'P'(주차)로 변속하면 주차 중 후방 뷰가 꺼집니다.
  2. 후방 모니터 화면의 백 버튼을 누르면 꺼집니다.
  3. 'N' 또는 'D' 상태에서 차량 속도가 10 km/h보다 빠르면 꺼집니다.

제한 사항

  • 카메라 영상은 실제 거리감과 다를 수 있습니다.
  • 카메라 렌즈가 오염되면 영상 표시 품질이 저하될 수 있습니다.

주의 사항

  • 후방 모니터 영상만 의존하지 말고 주변을 직접 확인하십시오.
  • 렌즈 표면을 깨끗하게 유지하십시오.

FAQ

후방 모니터 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
