디지털 키/인증

NFC 카드 키

NFC 카드 키를 이용해 도어 잠금/잠금 해제와 차량 인증을 지원하는 디지털 키 기능입니다.

기능 개요

  1. NFC 카드 키를 이용해 도어 잠금/잠금 해제와 차량 인증을 지원하는 디지털 키 기능입니다.

사용 전 조건

  1. 디지털 키 사양이 적용된 차량이어야 합니다.
  2. 차량에 NFC 카드 키가 등록되어 있어야 합니다.

설정 방법

  1. 인포테인먼트 시스템의 디지털 키 또는 사용자 프로필 관련 메뉴에서 등록 및 연동 상태를 확인합니다.

사용 방법

  1. 등록된 NFC 카드 키를 도어 핸들 또는 차량에서 안내하는 인증 위치에 접촉해 사용합니다.
  2. 프로필 개인화는 스마트폰 키와 다르게 작동하며, 카드 키는 최근 작동한 프로필 기준으로 안내됩니다.

해제 방법

  1. 디지털 키 관련 메뉴에서 등록된 키를 삭제하거나 연동을 해제합니다.

작동 제한 상황

  1. NFC 카드 키로는 프로필 연동을 할 수 없습니다.
  2. 카드 키 등록 여부와 차량 사양에 따라 사용 가능 범위가 달라집니다.

주의 사항

  1. 카드 키 분실 시 차량의 디지털 키 등록 상태를 확인하고 필요한 조치를 하십시오.

관련 기능

  • Hyundai Digital Key 2
  • 스마트 키
