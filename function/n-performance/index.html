N 전용 주행/성능 기능

현대 N 차량에서 성능 주행 모드, 표시, 전동화 성능 설정을 제공하는 전용 기능군입니다.

기능 개요

  • 현대 N 차량에서 성능 주행 모드, 표시, 전동화 성능 설정을 제공하는 전용 기능군입니다.

사용 전 조건

  • N 차량이어야 합니다.
  • 주행 환경과 노면 상태가 안전해야 합니다.

사용 방법

  • 스티어링 휠 N 버튼 또는 드라이브 모드 버튼을 누릅니다.
  • N 모드 또는 사용자 설정 모드를 선택합니다.
  • 클러스터 성능 표시와 출력/서스펜션/회생 설정을 확인합니다.

해제 방법

  • 드라이브 모드를 Normal 또는 Eco로 변경합니다.

작동 제한 상황

  • 일부 성능 기능은 배터리 온도, 주행 모드, 안전 조건에 따라 제한됩니다.

주의 사항

  • 공도에서는 법규와 안전을 우선하세요.
