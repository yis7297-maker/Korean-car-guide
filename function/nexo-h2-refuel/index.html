FCEV / 수소 충전

수소 충전 / 충전구 개폐

디 올 뉴 넥쏘의 수소 충전구를 열고 충전소 노즐을 연결해 수소를 충전합니다.

공식 자료 검증 완료

기능 설명

디 올 뉴 넥쏘의 수소 충전구를 열고 충전소 노즐을 연결해 수소를 충전합니다.

사용 전 조건

  • 차량을 수소 충전기 앞에 정차하고 P(주차) 상태로 전환합니다.
  • 시동을 끄고 충전소 안내를 확인합니다.

설정 방법

  1. FCEV → 에너지 또는 수소전기차 설정

사용 방법

  1. 운전석 주변 충전구 열림 버튼 또는 차량 화면 메뉴로 충전구를 엽니다.
  2. 충전구 캡을 열고 수소 충전 노즐을 체결합니다.
  3. 충전기 시작 버튼을 누르고 충전량 표시를 확인합니다.
  4. 충전 완료 후 노즐을 분리하고 충전구 캡과 도어를 닫습니다.

해제 방법

  1. 충전기 정지 버튼을 누르거나 충전 완료 후 노즐을 분리합니다.

제한 사항

  • 충전소 압력, 외기 온도, 탱크 상태에 따라 충전량과 시간이 달라집니다.

주의 사항

  • 수소 충전 중 흡연, 화기 사용, 임의 분리는 금지됩니다.

관련 기능

연결된 관련 기능이 없습니다.
