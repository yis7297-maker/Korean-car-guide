FCEV / 에코 주행

에코 코칭 / 주행 효율 안내

수소전기차 주행 효율을 높이도록 가속, 감속, 공조 사용 등 운전 습관을 안내합니다.

공식 자료 검증 완료

기능 설명

수소전기차 주행 효율을 높이도록 가속, 감속, 공조 사용 등 운전 습관을 안내합니다.

사용 전 조건

  • 클러스터 또는 인포테인먼트 에코 메뉴 접근이 가능해야 합니다.

설정 방법

  1. FCEV → 에너지 또는 수소전기차 설정

사용 방법

  1. 클러스터 또는 FCEV 정보 화면에서 에코/효율 메뉴를 선택합니다.
  2. 주행 점수, 회생 정보, 에너지 사용량을 확인합니다.
  3. 급가속과 급제동을 줄이고 안내에 맞춰 주행합니다.

해제 방법

  1. 클러스터 표시 항목을 변경하거나 에코 안내 화면을 닫습니다.

제한 사항

  • 점수와 안내는 실제 교통 상황과 도로 조건에 따라 달라집니다.

주의 사항

  • 효율 안내보다 교통 흐름과 안전 운전을 우선하세요.

관련 기능

FCEV 정보 화면 / 에너지 모니터스마트 회생제동 / 회생제동 조절
