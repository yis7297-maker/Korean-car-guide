EV 전용 기능

충전 전류 설정

완속 충전 또는 휴대용 충전 시 충전 전류를 설정하는 기능입니다.

기능 개요

  1. 완속 충전 또는 휴대용 충전 시 충전 전류를 설정하는 기능입니다.

사용 전 조건

  1. 완속 충전 또는 휴대용 충전 조건이어야 합니다.

설정 방법

  1. 인포테인먼트 시스템 홈 화면에서 Electric Vehicle > EV 충전 설정 > 완속 충전 설정을 선택합니다.

사용 방법

  1. 완속 충전 설정에서 충전 전류를 설정합니다.
  2. 휴대용 충전기 사용 시 제어 박스의 버튼을 2초 이상 길게 눌러 충전 전류를 설정합니다.

해제 방법

  1. 충전 설정에서 충전 전류를 변경하거나 충전을 종료합니다.

작동 제한 상황

  1. 공급 전원의 정격 전류와 충전기 사양에 따라 설정 가능한 전류가 제한됩니다.

주의 사항

  1. 가정용 전력의 충전 전압, 전류 및 부하 조건을 확인한 후 사용하십시오.

관련 기능

  • 충전 예약
  • 충전 상태 확인
