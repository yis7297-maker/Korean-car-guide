하이브리드 전용 주행 정보

스타리아 하이브리드 매뉴얼 기준 EV 모드 표시등은 모터를 이용해 주행하는 상태를 표시합니다.

기능 개요

  1. 스타리아 하이브리드 매뉴얼 기준 EV 모드 표시등은 모터를 이용해 주행하는 상태를 표시합니다.

사용 전 조건

  1. 하이브리드 차량 전원이 켜져 있고 클러스터 표시가 가능해야 합니다.

설정 방법

  1. 클러스터 주행 정보 또는 하이브리드 관련 표시 항목을 확인합니다.

사용 방법

  1. 클러스터에서 EV 모드 표시등 또는 하이브리드 주행 정보를 확인합니다.
  2. 정차 중 엔진 정지 또는 주행 중 모터 단독 구동 상태를 확인합니다.

해제 방법

  1. 클러스터 표시 항목을 다른 화면으로 변경합니다.

작동 제한 상황

  1. 표시 정보는 주행 조건과 시스템 상태에 따라 달라집니다.

주의 사항

  1. 표시 정보는 참고용이며 안전 주행을 우선하십시오.

관련 기능

  • 하이브리드 에너지 흐름도
  • 하이브리드 스마트 회생제동
