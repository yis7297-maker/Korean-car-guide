완속 충전

코나 일렉트릭의 완속 충전기를 이용해 고전압 배터리를 충전하는 기능입니다.

기능 개요

  • 코나 일렉트릭의 완속 충전기를 이용해 고전압 배터리를 충전하는 기능입니다.

사용 전 조건

  • 완속 충전기와 차량 충전 커넥터를 사용할 수 있어야 합니다.
  • 충전 전 안전 주의 사항을 확인해야 합니다.

설정 방법

  • 필요 시 Electric Vehicle > EV 충전 설정 > 완속 충전 설정에서 충전 관련 항목을 확인합니다.

사용 방법

  • 완속 충전기 사용하기 절차에 따라 충전 커넥터를 차량에 연결합니다.
  • 충전 상태 표시 정보를 확인합니다.
  • 충전이 완료되면 매뉴얼 안내에 따라 충전 커넥터를 분리합니다.

해제 방법

  • 충전 완료 후 충전 커넥터를 분리합니다.
  • 필요 시 충전 즉시 중단 기능을 사용합니다.

작동 제한 상황

  • 충전 설비, 케이블 정격, 충전 전류 설정, 차량 배터리 상태에 따라 충전 시간이 달라질 수 있습니다.

주의 사항

  • 젖은 손이나 손상된 충전 케이블로 충전하지 않습니다.

관련 기능

  • EV 충전 예약 / 배터리 컨디셔닝 / V2L
  • 충전 전류 설정
  • 충전 커넥터 잠금 모드
