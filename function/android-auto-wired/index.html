Android Auto

Android 휴대폰을 USB로 연결해 지도, 통화, 음악, 메시지 앱을 차량 화면에서 사용합니다.

공식 자료 검증 완료

기능 설명

Android 휴대폰을 USB로 연결해 지도, 통화, 음악, 메시지 앱을 차량 화면에서 사용합니다.

사용 전 조건

  • Android Auto 지원 휴대폰과 데이터 통신 가능한 USB 케이블이 필요합니다.
  • 휴대폰의 Android Auto 권한을 허용해야 합니다.

설정 방법

  1. 홈 → 설정 → 기기 연결 또는 시스템

사용 방법

  1. 차량을 정차하고 USB 데이터 포트에 휴대폰을 연결합니다.
  2. 휴대폰에서 Android Auto 권한과 약관을 확인합니다.
  3. 차량 화면에서 Android Auto 아이콘을 선택합니다.
  4. 지도와 미디어 앱이 표시되는지 확인합니다.

해제 방법

  1. USB 케이블을 분리합니다.
  2. 차량 기기 연결 설정에서 휴대폰을 삭제합니다.

제한 사항

  • 케이블 품질, 앱 권한, 휴대폰 제조사 설정에 따라 연결이 실패할 수 있습니다.

주의 사항

  • 주행 중 휴대폰 직접 조작은 피하세요.

관련 기능

Android Auto 무선 연결Apple CarPlay
