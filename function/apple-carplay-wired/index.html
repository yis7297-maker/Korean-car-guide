Apple CarPlay

iPhone을 USB로 연결해 지도, 통화, 음악, 메시지 앱을 차량 화면에서 사용합니다.

공식 자료 검증 완료

기능 설명

iPhone을 USB로 연결해 지도, 통화, 음악, 메시지 앱을 차량 화면에서 사용합니다.

사용 전 조건

  • iPhone의 Siri와 CarPlay가 활성화되어 있어야 합니다.
  • 데이터 통신 가능한 USB 케이블을 준비합니다.

설정 방법

  1. 홈 → 설정 → 기기 연결 또는 시스템

사용 방법

  1. 차량을 정차하고 USB 데이터 포트에 iPhone을 연결합니다.
  2. iPhone에서 CarPlay 사용 허용을 누릅니다.
  3. 차량 화면에서 Apple CarPlay 아이콘을 선택합니다.
  4. 연락처와 메시지 권한을 필요한 범위에서 허용합니다.

해제 방법

  1. USB 케이블을 분리합니다.
  2. 차량 기기 연결 설정에서 해당 iPhone을 삭제합니다.

제한 사항

  • 케이블 불량, iOS 제한, 앱 권한 문제로 연결이 실패할 수 있습니다.

주의 사항

  • 주행 중 iPhone을 직접 조작하지 마세요.

관련 기능

Apple CarPlay 무선 연결Android Auto
