Android Auto 무선 연결

지원 차량에서 Wi-Fi와 Bluetooth로 Android Auto를 연결합니다.

공식 자료 검증 완료

기능 설명

지원 차량에서 Wi-Fi와 Bluetooth로 Android Auto를 연결합니다.

사용 전 조건

  • 차량과 휴대폰이 무선 Android Auto를 지원해야 합니다.
  • 휴대폰 Bluetooth와 Wi-Fi가 켜져 있어야 합니다.

설정 방법

  1. 홈 → 설정 → 기기 연결 또는 시스템

사용 방법

  1. 차량 기기 연결 메뉴에서 새 기기 추가를 선택합니다.
  2. 휴대폰 Bluetooth 목록에서 차량 이름을 선택합니다.
  3. 인증 번호를 확인하고 페어링합니다.
  4. 휴대폰에서 Android Auto 무선 연결을 허용합니다.

해제 방법

  1. 차량 기기 연결 메뉴에서 해당 휴대폰을 삭제하거나 Android Auto 자동 실행을 끕니다.

제한 사항

  • 무선 간섭, 휴대폰 배터리 절약, 제조사 설정에 따라 재연결이 지연될 수 있습니다.

주의 사항

  • 최초 연결은 정차 상태에서 진행하세요.

관련 기능

Android AutoApple CarPlay 무선 연결
