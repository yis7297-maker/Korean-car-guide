메모리 주차 / Memory Parking

공식 매뉴얼에서 메모리 주차로 명시된 차량의 저장·재생형 주차 기능입니다.

기능 개요

  • 공식 매뉴얼에서 메모리 주차로 명시된 차량의 저장·재생형 주차 기능입니다.

사용 전 조건

  • 해당 기능이 적용된 차량이어야 하며 차량 화면의 안내 조건을 충족해야 합니다.

설정 방법

  • 공식 매뉴얼에 표시된 주차 보조 메뉴에서 메모리 주차를 선택합니다.

사용 방법

  • 차량 화면의 안내에 따라 경로 저장을 시작합니다.
  • 저장 완료 안내를 확인합니다.
  • 재생 시 화면 안내에 따라 차량 주변을 확인하며 기능을 실행합니다.

해제 방법

  • 브레이크 조작 또는 화면의 취소·종료 항목으로 종료합니다.

작동 제한 상황

  • 공식 매뉴얼에서 지원이 명시된 차량에만 사용할 수 있습니다.

주의 사항

  • 카메라 영상과 경고는 주변 확인을 대신하지 않습니다. 운전자가 직접 차량 주변을 확인하고 제동과 조향을 수행해야 합니다.

관련 기능

  • 메모리 리버스 어시스트 / MRA
