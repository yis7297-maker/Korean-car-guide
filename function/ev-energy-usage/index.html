EV 전용 기능

에너지 사용량

주행, 공조, 전장 장치가 사용한 고전압 배터리 에너지 비율과 효율 정보를 표시합니다.

기능 개요

  • 아이오닉 6 매뉴얼 기준 에너지 정보 화면에서는 전력 소비량과 연비 이력을 확인할 수 있습니다.

설정 방법

  • 인포테인먼트 시스템 홈 화면에서 Electric Vehicle > 에너지 정보를 선택합니다.

사용 방법

  • 에너지 정보 화면에서 전력 소비량을 선택합니다.
  • 차량 내부 시스템별 현재 전력 소비량을 확인합니다.

작동 제한 상황

  • 표시 정보는 주행 상태와 차량 시스템 사용 상태에 따라 달라집니다.

주의 사항

  • 에너지 정보는 참고용이며 주행 가능 거리와 충전 계획을 함께 확인하십시오.

관련 기능

  • EV Route Planner
  • 배터리 상태 확인
