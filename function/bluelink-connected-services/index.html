커넥티드 서비스

Bluelink

차량 통신 기반으로 원격 제어, 상태 조회, 안전/편의 서비스를 제공하는 현대 커넥티드 서비스입니다.

기능 개요

  • 차량 통신 기반으로 원격 제어, 상태 조회, 안전/편의 서비스를 제공하는 현대 커넥티드 서비스입니다.

사용 전 조건

  • Bluelink 지원 차량이어야 하며 서비스 가입과 앱 로그인이 필요합니다.

설정 방법

  • 인포테인먼트 시스템과 Bluelink 앱에서 서비스 가입 및 차량 등록 상태를 확인합니다.

사용 방법

  • Bluelink 앱 또는 차량 인포테인먼트의 Bluelink 메뉴를 이용합니다.
  • 원격 시동, 원격 공조, 차량 상태 조회 등 제공되는 메뉴를 선택합니다.

해제 방법

  • 앱 또는 차량 메뉴에서 실행 중인 원격 기능을 중지하거나 서비스 설정을 변경합니다.

작동 제한 상황

  • 통신 음영 지역, 차량 상태, 서비스 가입 상태에 따라 일부 기능이 제한될 수 있습니다.

주의 사항

  • 원격 기능 사용 전 차량 주변 안전과 실내/실외 환경을 확인하십시오.

관련 기능

  • 원격 시동
  • 원격 공조
