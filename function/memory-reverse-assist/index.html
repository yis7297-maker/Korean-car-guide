주차 편의 / 후진 보조

메모리 리버스 어시스트 / MRA

저속 전진 시 지나온 경로를 저장하고, 필요할 때 해당 경로를 따라 후진하도록 조향을 보조합니다.

공식 자료 검증 완료

기능 설명

저속 전진 시 지나온 경로를 저장하고, 필요할 때 해당 경로를 따라 후진하도록 조향을 보조합니다.

사용 전 조건

  • 더 뉴 그랜저의 해당 사양 적용 차량이어야 합니다.
  • 차량 주변과 저장할 전진 경로에 장애물이 없는지 확인합니다.
  • 차량 안내 화면에서 경로 저장 가능 상태인지 확인합니다.

설정 방법

  1. 설정 → 차량 → 운전자 보조 → 주차 안전

사용 방법

  1. 저속으로 전진하면서 주차 보조 화면에서 메모리 리버스 어시스트를 선택해 경로를 저장합니다.
  2. 후진이 필요한 위치에 정차한 뒤 기능을 활성화합니다.
  3. 화면 안내에 따라 브레이크와 주변을 확인하며 후진 경로 재생을 시작합니다.
  4. 차량이 저장 경로를 따라 조향하는 동안 운전자가 속도와 주변 안전을 계속 관리합니다.

해제 방법

  1. 브레이크를 밟거나 기능 종료를 선택해 경로 재생을 중지합니다.

제한 사항

  • 저장 경로를 벗어난 차량 위치, 경사가 큰 도로, 미끄러운 노면, 센서 오염, 장애물 감지 또는 시스템 안전 조건 미충족 시 제한되거나 종료될 수 있습니다.

주의 사항

  • 자동 주차나 자율주행 기능이 아니며, 충돌 방지와 제동 책임은 운전자에게 있습니다.

관련 기능

원격 스마트 주차 보조 2 / RSPA 2주차 충돌방지 보조
