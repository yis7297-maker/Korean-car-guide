기능 설명

시트 온열 또는 통풍을 단계별로 조절합니다.

사용 전 조건

  • 시동 또는 전원 ON 상태여야 합니다.
  • 해당 좌석에 열선/통풍 기능이 탑재되어야 합니다.

설정 방법

  1. 설정 → 차량 → 시트

사용 방법

  1. 센터 콘솔 또는 공조 화면의 시트 열선/통풍 버튼을 누릅니다.
  2. 1~3단계 중 원하는 세기를 선택합니다.
  3. 작동 아이콘과 단계 표시를 확인합니다.

해제 방법

  1. 같은 버튼을 반복해서 눌러 OFF로 전환합니다.

제한 사항

  • 뒷좌석 지원 여부와 단계 수는 차종/트림별로 다릅니다.

주의 사항

  • 감각이 둔한 탑승자는 열선 사용 시 저온 화상에 주의하세요.
