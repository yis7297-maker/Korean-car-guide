원격 스마트 주차 보조 / RSPA

차량 센서로 주차공간과 주변 상황을 인식하고 스티어링 휠, 속도, 변속을 자동으로 제어해 원격 전/후진을 도와주는 기능입니다.

공식 검증 완료

기능 설명

차량 센서로 주차공간과 주변 상황을 인식하고 스티어링 휠, 속도, 변속을 자동으로 제어해 원격 전/후진을 도와주는 기능입니다.

사용 전 조건

  • 스마트 키를 휴대하고 차량 주변 안전을 직접 확인해야 합니다.
  • 원격 스마트 주차 보조가 작동하면 주차 거리 경고와 서라운드 뷰 모니터가 함께 작동합니다.
  • 경고 방식은 시동 'ON' 상태에서 설정할 수 있습니다.

설정 방법

  1. 설정 > 차량 > 운전자 보조 > 경고 방식 > 경고 음량

사용 방법

  1. 주차/뷰 버튼을 길게 눌러 원격 스마트 주차 보조를 켭니다.
  2. 인포테인먼트 시스템 화면의 안내문을 확인합니다.
  3. 스마트 키를 휴대하고 차량 밖으로 이동합니다.
  4. 스마트 키의 전진 또는 후진 버튼을 누르고 있는 동안 차량이 해당 방향으로 이동합니다.
  5. 버튼에서 손을 떼면 원격 전/후진 제어가 멈춥니다.

해제 방법

  1. 원격 전/후진 중 스마트 키 버튼에서 손을 떼면 제어가 정지됩니다.
  2. 기능 해제 안내가 표시되면 안내에 따라 차량이나 기능을 조작합니다.

제한 사항

  • 기능 이상 시 인포테인먼트 시스템 화면에 경고문이 표시되며 사용하지 말고 공식 서비스 거점에서 점검받아야 합니다.
  • 차량 주변의 사람, 동물, 물체를 운전자가 직접 확인해야 합니다.
  • 주차 공간과 주변 환경에 따라 원격 전/후진 기능이 제한될 수 있습니다.

주의 사항

  • 버튼 조작 중에는 항상 주변 상황을 직접 확인하십시오.
  • 차량을 직접 조작해야 할 때 탑승할 수 있도록 충분한 공간이 있는 곳에서 사용하십시오.

FAQ

원격 스마트 주차 보조 / RSPA 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
