무선 소프트웨어 업데이트 / OTA

차량 내 화면 또는 휴대폰 안내를 통해 준비된 소프트웨어 업데이트를 승인하고 진행하는 기능입니다.

공식 검증 완료

기능 설명

차량 내 화면 또는 휴대폰 안내를 통해 준비된 소프트웨어 업데이트를 승인하고 진행하는 기능입니다.

사용 전 조건

  • 무선 소프트웨어 업데이트 준비가 완료되어야 합니다.
  • 블루링크에 가입한 경우 사용할 수 있습니다.
  • 안전한 장소에 주차한 후 충분한 시간을 확보합니다.

설정 방법

해당 차량의 공식 자료에서 별도 안내를 확인하세요.

사용 방법

  1. 차량 내 화면이나 휴대폰에서 업데이트 준비 완료를 확인합니다.
  2. 소프트웨어 업데이트 승인 화면에서 시작을 누릅니다.
  3. 차량이 업데이트를 준비한 후 진행 및 완료 안내를 확인합니다.

해제 방법

  1. 소프트웨어 업데이트가 시작되면 취소할 수 없습니다.

제한 사항

  • 업데이트 중에는 시동을 걸고 주행할 수 없습니다.
  • 원격 시동을 포함한 원격 서비스를 사용할 수 없습니다.
  • OBD 단자에 별도 제품이 연결되면 업데이트 기능이 동작하지 않습니다.

주의 사항

  • 업데이트 중 후드를 열거나 배터리를 분리하지 마십시오.
  • 제어기나 소프트웨어를 임의로 교체하면 무선 업데이트 기능을 사용할 수 없습니다.
  • 디지털 키 기능 업데이트가 포함되면 디지털 키, 지문 인증, 얼굴 인식 도어 잠금/해제가 작동하지 않을 수 있습니다.

FAQ

ccNC OTA 업데이트 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
