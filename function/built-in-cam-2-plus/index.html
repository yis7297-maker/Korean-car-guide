빌트인 캠 2 Plus

공식 txt에서 Plus 전용 절차는 별도 확인되지 않았으므로, 빌트인 캠 공통 챕터에서 확인된 녹화 설정, 수동 녹화, 영상 확인, 앱 연결 절차만 제공합니다.

기능 개요

  • 공식 txt에서 Plus 전용 절차는 별도 확인되지 않았으므로, 빌트인 캠 공통 챕터에서 확인된 녹화 설정, 수동 녹화, 영상 확인, 앱 연결 절차만 제공합니다.

사용 전 조건

  • 빌트인 캠 계열 사양이 적용된 차량이어야 합니다.
  • 인포테인먼트 시스템에서 녹화 조건을 선택해야 합니다.
  • SD 메모리가 정상 삽입되어 있고 충분한 가용 용량이 있어야 합니다.
  • 주차 중 녹화는 정품 12V 리튬 보조 배터리 장착 조건에서 사용할 수 있습니다.

설정 방법

  • 인포테인먼트 시스템 홈 화면에서 빌트인 캠을 선택합니다.
  • 빌트인 캠 설정 메뉴에서 녹화 조건과 주행 정보 표시 항목을 설정합니다.
  • 휴대폰 앱 연결이 필요한 경우 빌트인 캠 화면의 휴대폰 연결 메뉴에서 Wi‑Fi 활성화를 선택합니다.

사용 방법

  • 설정한 조건에 따라 주행 중 또는 주차 중 녹화가 작동합니다.
  • 수동 녹화 버튼을 짧게 누르면 이벤트 영상이 저장됩니다.
  • 수동 녹화 버튼을 길게 누르면 타임랩스 녹화가 시작되고 다시 길게 누르면 종료됩니다.
  • 녹화 영상을 확인하려면 인포테인먼트 시스템 홈 화면에서 빌트인 캠을 선택합니다.

해제 방법

  • 녹화 설정을 해제하면 해당 녹화가 작동하지 않습니다.
  • 수동 녹화 중 시동을 끄거나 빌트인 캠 메뉴를 선택하면 녹화가 종료됩니다.
  • 주차 녹화 설정 시간이 초과되거나 보조 배터리 충전량이 부족하면 주차 중 녹화가 종료됩니다.

작동 제한 상황

  • Plus 전용 화면명이나 별도 조작 절차는 현재 txt 본문에서 확인되지 않았습니다.
  • 빌트인 캠 메뉴 사용 중에는 녹화 기능이 작동하지 않습니다.
  • SD 메모리 미삽입, 고장, 수명 종료 상태에서는 녹화할 수 없습니다.
  • 소프트웨어 업데이트 중에는 빌트인 캠 녹화 기능이 작동하지 않습니다.

주의 사항

  • 중요 영상은 자동 삭제될 수 있으므로 다운로드하거나 보관함에 저장하십시오.
  • 음성 녹음 사용 시 탑승자 동의를 구하십시오.
  • 카메라 렌즈와 유리창을 깨끗하게 유지하십시오.
  • Wi‑Fi 비밀번호가 유출되지 않도록 관리하십시오.
