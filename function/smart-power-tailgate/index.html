스마트 파워 테일게이트

팰리세이드 매뉴얼 기준 파워 테일게이트는 스마트 키, 차량 버튼, 테일게이트 버튼으로 전동 개폐할 수 있으며 스마트 테일게이트 설정 시 후방 감지 영역 접근으로 자동 열림을 지원합니다.

기능 개요

  1. 팰리세이드 매뉴얼 기준 파워 테일게이트는 스마트 키, 차량 버튼, 테일게이트 버튼으로 전동 개폐할 수 있으며 스마트 테일게이트 설정 시 후방 감지 영역 접근으로 자동 열림을 지원합니다.

사용 전 조건

  1. 테일게이트 주변에 사람이나 장애물이 없는지 확인해야 합니다.
  2. 스마트 테일게이트는 모든 도어가 닫히고 잠긴 상태에서 일정 시간이 지난 뒤 작동 조건을 만족해야 합니다.
  3. 스마트 키를 휴대하고 차량 뒤쪽 감지 영역에 접근해야 합니다.

설정 방법

  1. 시동 ON 상태에서 인포테인먼트 시스템의 설정 > 차량 > 도어 > 스마트 테일게이트를 선택해 기능을 켜거나 끕니다.
  2. 파워 테일게이트 열림 높이와 열림 속도는 차량 설정의 도어/테일게이트 메뉴에서 설정합니다.

사용 방법

  1. 스마트 키의 테일게이트 열림/닫힘 버튼을 길게 누르면 테일게이트가 작동합니다.
  2. 차량 실내의 테일게이트 버튼 또는 테일게이트 외부 핸들 스위치를 눌러 열 수 있습니다.
  3. 열린 테일게이트는 스마트 키 버튼, 실내 버튼, 테일게이트 닫힘 버튼으로 닫습니다.
  4. 스마트 테일게이트 설정 후 스마트 키를 휴대하고 후방 감지 영역에 머물면 경고 후 테일게이트가 자동으로 열립니다.

해제 방법

  1. 스마트 테일게이트 감지 또는 경보 중 스마트 키 버튼이나 테일게이트 작동 버튼을 누르면 작동이 중지됩니다.
  2. 설정 > 차량 > 도어 > 스마트 테일게이트에서 기능을 해제합니다.

작동 제한 상황

  1. 영하의 기온, 주파수 혼선 환경, 차량 주변 장애물, 스마트 키 위치 조건에 따라 작동이 제한될 수 있습니다.
  2. 차량 안에 스마트 키가 있거나 조건이 맞지 않으면 스마트 테일게이트가 작동하지 않을 수 있습니다.

주의사항

  1. 테일게이트 작동 전 주변 사람과 물체를 확인하십시오.
  2. 차 안에 어린이나 애완동물만 남겨둔 상태로 차량을 떠나지 마십시오.
  3. 세차 시 스마트 테일게이트 기능을 중지하십시오.
