스마트 파워 테일게이트

쏘나타 매뉴얼 기준 파워 트렁크는 스마트 키 또는 차량 버튼으로 트렁크를 전동 개폐하고, 스마트 트렁크 설정 시 후방 접근 감지로 자동 열림을 지원합니다.

기능 개요

  1. 쏘나타 매뉴얼 기준 파워 트렁크는 스마트 키 또는 차량 버튼으로 트렁크를 전동 개폐하고, 스마트 트렁크 설정 시 후방 접근 감지로 자동 열림을 지원합니다.

사용 전 조건

  1. 트렁크 주변에 사람이나 장애물이 없는지 확인해야 합니다.
  2. 스마트 트렁크는 모든 도어가 닫히고 잠긴 상태에서 작동 조건을 만족해야 합니다.
  3. 스마트 키를 휴대하고 차량 뒤쪽 감지 영역에 접근해야 합니다.

설정 방법

  1. 시동 ON 상태에서 인포테인먼트 시스템의 설정 > 차량 > 도어 > 스마트 트렁크를 선택해 기능을 켜거나 끕니다.

사용 방법

  1. 스마트 키의 트렁크 열림/닫힘 버튼을 길게 눌러 트렁크를 작동합니다.
  2. 차량의 트렁크 버튼 또는 외부 핸들 스위치로 트렁크를 열 수 있습니다.
  3. 스마트 트렁크 설정 후 스마트 키를 휴대하고 후방 감지 영역에 머물면 경고 후 트렁크가 자동으로 열립니다.

해제 방법

  1. 스마트 트렁크 감지 또는 경보 중 스마트 키 버튼이나 트렁크 작동 버튼을 누르면 작동이 중지됩니다.
  2. 설정 > 차량 > 도어 > 스마트 트렁크에서 기능을 해제합니다.

작동 제한 상황

  1. 영하의 기온, 주파수 혼선 환경, 차량 주변 장애물, 스마트 키 위치 조건에 따라 작동이 제한될 수 있습니다.

주의 사항

  1. 트렁크 작동 전 주변 사람과 물체를 확인하십시오.
  2. 차 안에 어린이나 애완동물만 남겨둔 상태로 차량을 떠나지 마십시오.
