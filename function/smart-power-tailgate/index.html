스마트 파워 테일게이트

스마트키를 소지하고 차량 뒤쪽에 접근하거나 버튼을 눌러 테일게이트를 전동 개폐합니다.

공식 자료 검증 완료

기능 설명

스마트키를 소지하고 차량 뒤쪽에 접근하거나 버튼을 눌러 테일게이트를 전동 개폐합니다.

사용 전 조건

  • 스마트키를 휴대해야 합니다.
  • 차량이 잠금 상태이고 테일게이트 주변 공간이 충분해야 합니다.

설정 방법

  1. 홈 → 설정 → 차량 → 해당 기능 메뉴

사용 방법

  1. 차량 설정에서 스마트 파워 테일게이트 기능을 켭니다.
  2. 스마트키를 소지하고 후방 감지 영역에 머뭅니다.
  3. 경고음 또는 비상등 안내 후 테일게이트가 열리는지 확인합니다.
  4. 닫을 때는 테일게이트 버튼 또는 스마트키 버튼을 누릅니다.

해제 방법

  1. 차량 설정에서 스마트 테일게이트를 끕니다.
  2. 작동 중 버튼을 누르면 정지하거나 반전될 수 있습니다.

제한 사항

  • 키 위치, 전파 간섭, 낮은 천장, 장애물, 경사로에서 제한될 수 있습니다.

주의 사항

  • 열림 범위 안에 사람이나 물건이 없는지 확인하세요.

관련 기능

연결된 관련 기능이 없습니다.
