공식 자료 검증 완료

기능 설명

운전석과 동승석 온도를 따로 맞추고 풍량과 송풍 방향을 자동 제어합니다.

사용 전 조건

  • 시동 또는 전원 ON 상태여야 합니다.
  • 공조 패널이나 공조 화면이 사용 가능해야 합니다.

설정 방법

  1. 공조 화면 → 공조 설정

사용 방법

  1. 공조 패널의 AUTO 버튼을 누릅니다.
  2. 운전석 온도 조절 노브/버튼으로 원하는 온도를 설정합니다.
  3. SYNC가 꺼진 차량은 동승석 온도를 따로 설정합니다.
  4. A/C와 외기/내기 상태를 화면에서 확인합니다.

해제 방법

  1. AUTO 버튼을 다시 누르거나 풍량/송풍 방향을 수동 조작합니다.
  2. OFF 버튼으로 공조를 끕니다.

제한 사항

  • 극저온/고온, 문 열림, 배터리 상태에 따라 성능이 달라질 수 있습니다.

주의 사항

  • 김서림이 생기면 전방 디포그를 우선 사용하세요.
