충전 커넥터 잠금 모드

완속 충전 중 충전 커넥터가 의도치 않게 분리되는 것을 방지하기 위해 잠금 방식을 설정합니다.

기능 개요

  • 완속 충전 중 충전 커넥터가 의도치 않게 분리되는 것을 방지하기 위해 잠금 방식을 설정합니다.

사용 전 조건

  • 충전 커넥터가 차량에 연결되어 있거나 완속 충전 설정을 변경할 수 있어야 합니다.

설정 방법

  • 인포테인먼트 시스템 홈 화면에서 Electric Vehicle > EV 충전 설정 > 완속 충전 설정을 선택합니다.

사용 방법

  • 완속 충전 설정의 상세 기능에서 상시 잠금, 충전 중 잠금, 사용 안 함 중 하나를 선택합니다.
  • 급속 충전 및 전기 사용(V2L) 기능 사용 중에는 설정과 관계없이 커넥터가 자동으로 잠깁니다.

해제 방법

  • 사용 안 함을 선택하거나 충전 종료 후 도어 잠금을 해제해 커넥터 잠금을 해제합니다.

작동 제한 상황

  • 급속 충전 및 V2L 사용 중에는 설정과 관계없이 커넥터가 자동으로 잠깁니다.

주의 사항

  • 충전 커넥터 잠금 해제 버튼을 누르지 않은 상태로 강제로 분리하지 마십시오.

관련 기능

  • 충전 예약
  • V2L
