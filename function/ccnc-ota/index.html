차량 소프트웨어와 인포테인먼트 기능을 무선으로 업데이트합니다.

공식 자료 검증 완료

기능 설명

차량 소프트웨어와 인포테인먼트 기능을 무선으로 업데이트합니다.

사용 전 조건

  • 차량 통신과 배터리 상태가 정상이어야 합니다.
  • 업데이트 알림이 표시되어야 합니다.

설정 방법

  1. 홈 → 커넥티드 서비스 또는 공식 앱

사용 방법

  1. 업데이트 알림이 표시되면 내용을 확인합니다.
  2. 안전한 장소에 정차하고 P(주차) 상태로 전환합니다.
  3. 화면의 업데이트 시작 버튼을 누릅니다.
  4. 완료 메시지가 나올 때까지 차량 안내를 따릅니다.

해제 방법

  1. 업데이트 예약 또는 나중에 하기를 선택합니다.

제한 사항

  • 통신 상태, 배터리 잔량, 차량 전원 조건에 따라 지연될 수 있습니다.

주의 사항

  • 업데이트 중 전원을 임의로 끄지 마세요.

관련 기능

연결된 관련 기능이 없습니다.
