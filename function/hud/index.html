헤드업 디스플레이

속도, 내비게이션, 주행 보조 정보를 앞유리 영역에 표시합니다.

공식 자료 검증 완료

기능 설명

속도, 내비게이션, 주행 보조 정보를 앞유리 영역에 표시합니다.

사용 전 조건

  • HUD 탑재 차량이어야 합니다.
  • 시동 또는 전원 ON 상태여야 합니다.

설정 방법

  1. 홈 → 설정 → 차량 → 해당 기능 메뉴

사용 방법

  1. 차량 설정 > 디스플레이 > 헤드업 디스플레이 메뉴를 엽니다.
  2. HUD 표시를 켭니다.
  3. 높이, 밝기, 회전, 표시 항목을 조절합니다.

해제 방법

  1. 헤드업 디스플레이 메뉴에서 표시를 끕니다.

제한 사항

  • 편광 선글라스, 강한 햇빛, 앞유리 상태에 따라 잘 보이지 않을 수 있습니다.

주의 사항

  • 주행 중 표시 설정을 오래 조작하지 마세요.

관련 기능

연결된 관련 기능이 없습니다.
