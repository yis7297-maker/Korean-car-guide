인포테인먼트

음성인식 시스템

스티어링 휠의 음성 인식 버튼을 이용해 차량 인포테인먼트 기능을 음성으로 조작하는 기능입니다.

기능 개요

  1. 스티어링 휠의 음성 인식 버튼을 이용해 차량 인포테인먼트 기능을 음성으로 조작하는 기능입니다.

사용 전 조건

  1. 인포테인먼트 시스템과 음성 인식 기능이 사용 가능한 상태여야 합니다.

설정 방법

  1. 스티어링 휠의 음성 인식 버튼 위치를 확인합니다.

사용 방법

  1. 스티어링 휠의 음성 인식 버튼을 누릅니다.
  2. 안내음 후 원하는 명령어를 말합니다.

해제 방법

  1. 음성 인식 종료 안내를 따르거나 다른 인포테인먼트 조작으로 종료합니다.

작동 제한 상황

  1. 주변 소음, 발음, 시스템 상태에 따라 인식이 제한될 수 있습니다.

주의 사항

  1. 주행 중에는 화면 조작보다 안전 운전에 우선하십시오.

관련 기능

  • Bluelink
  • Android Auto
