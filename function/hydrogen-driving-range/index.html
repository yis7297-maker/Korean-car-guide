FCEV

수소 주행 가능 거리

현재 수소 잔량을 기준으로 계기판에 주행 가능 거리를 표시하는 기능입니다.

기능 개요

  • 현재 수소 잔량을 기준으로 계기판에 주행 가능 거리를 표시하는 기능입니다.

사용 전 조건

  • 차량 전원이 켜져 있고 수소 잔량 및 주행 정보가 표시 가능한 상태여야 합니다.

설정 방법

  • 계기판 또는 FCEV 관련 정보 화면에서 주행 가능 거리 정보를 확인합니다.

사용 방법

  • 계기판에 표시되는 주행 가능 거리를 확인합니다.
  • 주행 가능 거리는 현재 수소 잔량과 최근 주행 조건을 기반으로 표시됩니다.

해제 방법

  • 별도 해제 기능이 아니라 계기판 표시 정보입니다.

작동 제한 상황

  • 운전 습관, 전기 부하, 외부 환경, 고전압 배터리 에너지와 상태에 따라 표시 거리가 달라질 수 있습니다.

주의 사항

  • 표시값 보정이나 초기화가 실제 주행 가능 거리를 늘리는 것은 아닙니다.

관련 기능

  • 수소 탱크 정보
  • 수소 충전 / 충전구 개폐
  • FCEV 에너지 흐름도
