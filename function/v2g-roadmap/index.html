기능 설명

차량 배터리를 전력망과 양방향 연계하는 향후 기능입니다.

사용 전 조건

  • 해당 기능이 적용된 차종과 트림이어야 합니다.

설정 방법

  1. EV → 전력 공급

사용 방법

  1. 차량 화면 또는 전용 버튼에서 기능을 선택합니다.
  2. 화면 안내와 차량 상태를 확인한 뒤 사용합니다.

해제 방법

  1. 기능 화면에서 종료하거나 차량 안내에 따라 해제합니다.

제한 사항

  • 차종, 연식, 트림, 옵션과 차량 상태에 따라 지원 범위가 달라질 수 있습니다.

주의 사항

  • 주행 중에는 안전을 우선하고 복잡한 화면 조작을 피하세요.

관련 기능

V2H (향후)

적용 차량

현재 확정된 적용 차량 정보가 없습니다.
