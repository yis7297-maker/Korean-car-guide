원격 스마트 주차 보조 2 / RSPA 2

차량 안 또는 차량 밖에서 주차 공간 탐색과 주차·출차 조작을 보조합니다.

기능 개요

  • 차량 안 또는 차량 밖에서 주차 공간 탐색과 주차·출차 조작을 보조합니다.

사용 전 조건

  • 차량을 완전히 정차합니다.
  • 변속 위치를 P(주차)로 둡니다.
  • 시동을 유지하고 스마트키를 휴대합니다.
  • 차량 주변 사람과 장애물을 확인합니다.

설정 방법

  • 센터 콘솔의 주차/뷰 버튼과 화면의 원격 스마트 주차 보조 메뉴를 사용합니다.
  • 차량 밖에서는 스마트키의 원격 시동 및 전진·후진 버튼을 사용합니다.

사용 방법

  • 주차 보조 버튼 또는 주차 보조 메뉴를 실행합니다.
  • 화면 안내에 따라 차량 밖으로 이동합니다.
  • 스마트키의 원격 전진/후진 또는 원격 주차 버튼을 길게 누릅니다.
  • 차량 이동 방향과 주변 상태를 계속 확인합니다.
  • 주차 완료 후 버튼에서 손을 떼고 정지 상태를 확인합니다.

해제 방법

  • 스마트키 버튼에서 손을 떼면 차량 이동이 멈춥니다.
  • 브레이크 조작, 문 열림, 장애물 감지 시 중지될 수 있습니다.

작동 제한 상황

  • 경사로, 좁은 공간, 낮은 장애물, 센서 오염, 눈비에서는 제한됩니다.

주의 사항

  • 카메라 영상과 경고는 주변 확인을 대신하지 않습니다. 운전자가 직접 차량 주변을 확인하고 제동과 조향을 수행해야 합니다.

관련 기능

  • 주차 충돌방지 보조
