주차 편의

원격 스마트 주차 보조 2 / RSPA 2

스마트키 또는 차량 안내에 따라 저속 주차와 출차를 보조합니다.

공식 자료 검증 완료

기능 설명

스마트키 또는 차량 안내에 따라 저속 주차와 출차를 보조합니다.

사용 전 조건

  • 차량을 완전히 정차합니다.
  • 변속 위치를 P(주차)로 둡니다.
  • 시동을 유지하고 스마트키를 휴대합니다.
  • 차량 주변 사람과 장애물을 확인합니다.

설정 방법

  1. 설정 → 차량 → 운전자 보조 → 주차 안전

사용 방법

  1. 주차 보조 버튼 또는 주차 보조 메뉴를 실행합니다.
  2. 화면 안내에 따라 차량 밖으로 이동합니다.
  3. 스마트키의 원격 전진/후진 또는 원격 주차 버튼을 길게 누릅니다.
  4. 차량 이동 방향과 주변 상태를 계속 확인합니다.
  5. 주차 완료 후 버튼에서 손을 떼고 정지 상태를 확인합니다.

해제 방법

  1. 스마트키 버튼에서 손을 떼면 차량 이동이 멈춥니다.
  2. 브레이크 조작, 문 열림, 장애물 감지 시 중지될 수 있습니다.

제한 사항

  • 경사로, 좁은 공간, 낮은 장애물, 센서 오염, 눈비에서는 제한됩니다.

주의 사항

  • 차량 밖에서도 주변 안전 책임은 운전자에게 있습니다.
