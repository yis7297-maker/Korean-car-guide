실내 V2L

차량 내부에서 전기 사용(V2L) 기능을 활용해 전기 기기를 사용할 수 있는 기능입니다.

공식 검증 완료

기능 설명

차량 내부에서 전기 사용(V2L) 기능을 활용해 전기 기기를 사용할 수 있는 기능입니다.

사용 전 조건

  • 유틸리티 모드가 활성화되어 있습니다.
  • V2L 기능 사용 조건을 만족합니다.

설정 방법

  1. EV(Electric Vehicle) > EV 설정 > 유틸리티 모드

사용 방법

  1. 유틸리티 모드를 활성화합니다.
  2. 차량 내부에서 전기 사용(V2L) 기능을 활용합니다.

해제 방법

  1. 유틸리티 모드를 종료하거나 V2L 전기 사용을 중지합니다.

제한 사항

  • 배터리 방전 제한량 또는 차량 상태에 따라 사용이 제한될 수 있습니다.

주의 사항

  • 전기 사용 후에는 V2L 사용 상태를 종료하십시오.

FAQ

실내 V2L 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
