기능 설명

수소 잔량, 주행 가능 거리, 전력 흐름, 연료전지 작동 상태를 차량 화면에서 확인합니다.

사용 전 조건

  • 차량 전원 ON 상태여야 합니다.
  • FCEV 또는 에너지 메뉴 접근이 가능해야 합니다.

설정 방법

  1. FCEV → 에너지 또는 수소전기차 설정

사용 방법

  1. 홈 화면에서 FCEV 또는 에너지 정보 메뉴를 선택합니다.
  2. 수소 충전량과 주행 가능 거리를 확인합니다.
  3. 에너지 흐름도에서 연료전지, 배터리, 모터의 흐름을 확인합니다.
  4. 에너지 모니터에서 주행 효율과 사용 정보를 확인합니다.

해제 방법

  1. 홈 버튼이나 다른 메뉴를 선택해 화면을 닫습니다.

제한 사항

  • 주행 가능 거리는 운전 습관, 온도, 공조 사용, 도로 조건에 따라 달라집니다.

주의 사항

  • 잔여 주행 가능 거리에 여유를 두고 충전 계획을 세우세요.
