충전 상태 확인

충전 중 차량의 충전량, 충전 상태, 주행 가능 거리 관련 정보를 확인하는 기능입니다.

공식 검증 완료

기능 설명

충전 중 차량의 충전량, 충전 상태, 주행 가능 거리 관련 정보를 확인하는 기능입니다.

사용 전 조건

  • 충전 중이거나 EV 정보 화면에 진입합니다.

설정 방법

  1. EV(Electric Vehicle) 또는 에너지 정보 화면에서 확인합니다.

사용 방법

  1. EV 정보 또는 충전 관련 화면을 엽니다.
  2. 고전압 배터리 충전량과 주행 가능 거리 정보를 확인합니다.

해제 방법

해당 차량의 공식 자료에서 별도 안내를 확인하세요.

제한 사항

  • 충전소 상태에 따라 충전량과 주행 가능 거리가 달라질 수 있습니다.

주의 사항

  • 충전량이 부족한 상태로 차량을 장기간 보관하지 마십시오.

FAQ

충전 상태 확인 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
