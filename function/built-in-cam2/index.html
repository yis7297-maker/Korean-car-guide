빌트인 캠/녹화

빌트인 캠 2

공식 매뉴얼의 빌트인 캠 챕터 기준으로, 차량 전·후방 영상과 음성을 저장하고 녹화 영상 확인, 휴대폰 앱 연결, 소프트웨어 업데이트를 지원합니다.

기능 개요

  1. 공식 매뉴얼의 빌트인 캠 챕터 기준으로, 차량 전·후방 영상과 음성을 저장하고 녹화 영상 확인, 휴대폰 앱 연결, 소프트웨어 업데이트를 지원합니다.

사용 전 조건

  1. 빌트인 캠 사양이 적용된 차량이어야 합니다.
  2. 녹화 조건은 인포테인먼트 시스템의 빌트인 캠 설정에서 선택해야 합니다.
  3. 64GB 이상의 SD 메모리가 정상 삽입되어 있고 가용 용량이 32GB 이상 남아 있어야 합니다.
  4. 주차 중 녹화는 정품 12V 리튬 보조 배터리 장착 조건에서 사용할 수 있습니다.

설정 방법

  1. 인포테인먼트 시스템 홈 화면에서 빌트인 캠을 선택합니다.
  2. 빌트인 캠 설정 메뉴에서 녹화 조건을 설정합니다.
  3. 주행 중 녹화, 주차 중 녹화, 음성 녹음 여부를 선택합니다.
  4. 휴대폰 앱 연결이 필요하면 빌트인 캠 화면의 휴대폰 연결 메뉴에서 Wi‑Fi 활성화를 선택합니다.

사용 방법

  1. 설정한 녹화 조건에 따라 주행 중 또는 주차 중 녹화가 작동합니다.
  2. 수동 녹화 버튼을 짧게 눌러 이벤트 영상을 저장합니다.
  3. 수동 녹화 버튼을 길게 눌러 타임랩스 녹화를 시작하고, 다시 길게 눌러 종료합니다.
  4. 녹화 영상을 보려면 인포테인먼트 시스템 홈 화면에서 빌트인 캠을 선택합니다.
  5. 휴대폰 앱으로 확인하려면 마이현대 또는 MY GENESIS 앱에서 빌트인 캠 연결하기를 선택하고 Wi‑Fi로 연결합니다.

해제 방법

  1. 녹화 설정을 해제하면 해당 녹화가 작동하지 않습니다.
  2. 수동 녹화 중 시동을 끄거나 빌트인 캠 메뉴에 진입하면 녹화가 종료됩니다.
  3. 주차 중 녹화는 설정 시간이 초과되거나 보조 배터리 충전량이 부족하면 종료됩니다.

작동 제한 상황

  1. 빌트인 캠 메뉴 사용 중에는 녹화가 작동하지 않습니다.
  2. SD 메모리 미삽입, 고장, 수명 종료 상태에서는 녹화할 수 없습니다.
  3. 휴대폰 앱은 휴대폰 기종이나 OS 버전에 따라 정상 작동하지 않을 수 있습니다.
  4. 업데이트 중에는 빌트인 캠 녹화 기능이 작동하지 않습니다.

주의 사항

  1. SD 메모리를 작동 중 제거하면 저장 파일 또는 파일 시스템이 손상될 수 있습니다.
  2. 저장 공간이 부족하면 오래된 파일부터 자동 삭제됩니다.
  3. 카메라 렌즈나 전·후면 유리창 오염 시 영상 품질이 저하될 수 있습니다.
  4. Wi‑Fi 연결 시 휴대폰의 모바일 네트워크 사용이 중단되므로 필요하면 Wi‑Fi 연결을 해제하십시오.
