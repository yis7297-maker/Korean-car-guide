후방 모니터

후진 시 후방 카메라 영상을 화면에 표시해 차량 뒤쪽 확인을 돕습니다.

기능 개요

  • 후진 시 후방 카메라 영상을 화면에 표시해 차량 뒤쪽 확인을 돕습니다.

사용 전 조건

  • 카메라 렌즈가 가려져 있지 않아야 합니다.

설정 방법

  • 변속 위치를 R(후진)로 전환하면 후방 영상이 표시됩니다. 지원 차량은 주차/뷰 버튼으로 영상을 표시할 수 있습니다.

사용 방법

  • R(후진)로 전환합니다.
  • 화면의 후방 영상과 가이드라인을 확인하면서 차량 주변을 직접 확인합니다.

해제 방법

  • R(후진) 이외의 위치로 변속하면 차량 조건에 따라 화면이 종료됩니다.
  • 주차/뷰 버튼 또는 화면의 종료 버튼으로 종료할 수 있습니다.

작동 제한 상황

  • 카메라 렌즈 오염, 어두운 장소, 강한 빛과 악천후에서는 영상이 선명하지 않을 수 있습니다.

주의 사항

  • 카메라 영상과 경고는 주변 확인을 대신하지 않습니다. 운전자가 직접 차량 주변을 확인하고 제동과 조향을 수행해야 합니다.

관련 기능

  • 후진 가이드
  • 서라운드 뷰 모니터 / SVM
