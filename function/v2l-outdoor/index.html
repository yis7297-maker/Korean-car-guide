실외 V2L

V2L 커넥터를 이용해 차량 외부에서 전기 기기를 사용할 수 있는 기능입니다.

공식 검증 완료

기능 설명

V2L 커넥터를 이용해 차량 외부에서 전기 기기를 사용할 수 있는 기능입니다.

사용 전 조건

  • V2L 커넥터와 충전 인렛이 정상입니다.
  • 전원 플러그 및 케이블이 손상되지 않았습니다.

설정 방법

  1. EV(Electric Vehicle) > 전기 사용(V2L) 설정에서 방전 제한량을 설정합니다.

사용 방법

  1. V2L 커넥터를 차량 충전 인렛에 연결합니다.
  2. 전기 기기를 V2L 커넥터에 연결해 사용합니다.
  3. 사용 후 V2L 커넥터 스위치를 눌러 전원을 끄고 도어 잠금 해제 후 분리합니다.

해제 방법

  1. 전원 스위치를 끄고 커넥터를 분리합니다.
  2. 과열 시 자동 차단될 수 있습니다.

제한 사항

  • V2L 커넥터 내부 회로 과열 시 자동 차단됩니다.
  • 손상/부식/느슨한 연결 시 사용하지 마십시오.

주의 사항

  • 젖은 손으로 만지지 마십시오.
  • V2L 커넥터를 임의 개조하거나 분해하지 마십시오.
  • V2L 커넥터로 차량을 충전하지 마십시오.

FAQ

실외 V2L 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
