메모리 시트

공식 자료 검증 완료

기능 설명

시트, 미러, HUD 등 일부 설정을 저장하고 버튼이나 프로필로 불러옵니다.

사용 전 조건

  • 차량이 정차 상태여야 합니다.
  • 메모리 기능 탑재 차량이어야 합니다.

설정 방법

  1. 설정 → 차량 → 시트

사용 방법

  1. 시트와 미러 위치를 맞춥니다.
  2. 도어 또는 시트 근처 SET 버튼을 누릅니다.
  3. 1 또는 2 메모리 버튼을 눌러 저장합니다.
  4. 불러올 때는 저장된 번호 버튼을 누릅니다.

해제 방법

  1. 다른 위치를 저장해 덮어쓰거나 프로필 연동을 해제합니다.

제한 사항

  • 차량 전원 상태와 도어 조건에 따라 이동이 제한될 수 있습니다.

주의 사항

  • 시트 이동 중 몸이나 물건이 끼이지 않게 주의하세요.

관련 기능

연결된 관련 기능이 없습니다.
