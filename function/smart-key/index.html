스마트 키

스마트 키로 도어 잠금/해제, 시동, 원격 시동 등 차량 편의 기능을 사용할 수 있습니다.

공식 검증 완료

기능 설명

스마트 키로 도어 잠금/해제, 시동, 원격 시동 등 차량 편의 기능을 사용할 수 있습니다.

사용 전 조건

  • 스마트 키를 휴대합니다.
  • 차량과 스마트 키가 작동 가능 거리 안에 있습니다.
  • 스마트 키 배터리 상태가 정상입니다.

설정 방법

해당 차량의 공식 자료에서 별도 안내를 확인하세요.

사용 방법

  1. 스마트 키를 휴대하거나 차 안에 둔 상태에서 시동을 걸 수 있습니다.
  2. 스마트 키로 도어 잠금/잠금 해제를 사용할 수 있습니다.
  3. 스마트 키가 인식되지 않을 경우 스마트 키로 시동 버튼을 직접 누릅니다.

해제 방법

해당 차량의 공식 자료에서 별도 안내를 확인하세요.

제한 사항

  • 작동 가능 거리(약 10 m 내외)를 벗어나면 작동하지 않을 수 있습니다.
  • 스마트 키 건전지가 약하면 작동하지 않을 수 있습니다.
  • 다른 송수신 장치 또는 금속성 물체 근처에서는 작동 거리나 기능이 제한될 수 있습니다.

주의 사항

  • 차에서 벗어날 때는 반드시 스마트 키를 휴대하십시오.
  • 스마트 키는 완전 방수가 아니므로 습기나 액체 유입에 주의하십시오.
  • 스마트 키는 한 차에 최대 2개까지 입력할 수 있습니다. 분실 시 하이테크센터나 블루핸즈에서 조치하십시오.

FAQ

스마트 키 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
