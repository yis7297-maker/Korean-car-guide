회생제동 단계 조절

패들 쉬프트 레버로 회생 제동량을 조절해 감속 시 고전압 배터리를 충전하는 기능입니다.

공식 검증 완료

기능 설명

패들 쉬프트 레버로 회생 제동량을 조절해 감속 시 고전압 배터리를 충전하는 기능입니다.

사용 전 조건

  • 전기차 주행 중입니다.
  • 회생 제동 단계 조절이 가능한 주행 상태입니다.

설정 방법

해당 차량의 공식 자료에서 별도 안내를 확인하세요.

사용 방법

  1. 패들 쉬프트 레버를 조작해 회생 제동량을 조절합니다.
  2. 클러스터의 회생 제동 단계 표시등에서 설정 단계를 확인합니다.
  3. 급감속 시에는 회생 제동 기능이 제한될 수 있습니다.

해제 방법

  1. 패들 쉬프트 조작 또는 주행 상태 변화에 따라 단계가 변경됩니다.

제한 사항

  • 급감속 시 회생 제동 기능이 제한됩니다.
  • 브레이크 시스템 이상 시 회생 제동 경고등이 켜질 수 있습니다.

주의 사항

  • 회생 제동 경고등이 켜지면 하이테크센터나 블루핸즈에서 점검을 받으십시오.

FAQ

회생제동 단계 조절 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
