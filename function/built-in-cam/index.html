빌트인 캠

차량에 장착된 빌트인 캠 카메라로 차량 전·후방의 영상과 음성을 저장하는 장치입니다. 주행/주차 중 상시 녹화, 이벤트 녹화, 수동 녹화, 타임랩스 녹화 항목을 차량 화면에서 관리합니다.

기능 개요

  1. 차량에 장착된 빌트인 캠 카메라로 차량 전·후방의 영상과 음성을 저장하는 장치입니다. 주행/주차 중 상시 녹화, 이벤트 녹화, 수동 녹화, 타임랩스 녹화 항목을 차량 화면에서 관리합니다.

사용 전 조건

  1. 빌트인 캠 사양이 적용된 차량이어야 합니다.
  2. 인포테인먼트 시스템에서 원하는 주행/주차 녹화 조건을 선택해야 합니다.
  3. 녹화하려면 64GB 이상의 SD 메모리가 정상 삽입되어 있고 가용 용량이 32GB 이상 남아 있어야 합니다.
  4. 주차 중 녹화는 정품 12V 리튬 보조 배터리 장착 조건에서 사용할 수 있습니다.

설정 방법

  1. 인포테인먼트 시스템 홈 화면에서 빌트인 캠을 선택합니다.
  2. 빌트인 캠 설정 메뉴를 선택합니다.
  3. 녹화 항목에서 주행 중 녹화, 주차 중 녹화, 음성 녹음 여부를 선택합니다.
  4. 주행 정보 표시 메뉴에서 시간, 속도, 기어, 방향지시등, 충격 감지 정도, GPS 위치 등 기록 항목을 선택합니다.

사용 방법

  1. 시동(ACC 또는 IGN 전원) ON 상태에서 주행 중 상시 녹화 또는 주행 중 이벤트 녹화를 설정하면 주행 중 녹화가 작동합니다.
  2. 시동(ACC 또는 IGN 전원) OFF 상태에서 주차 중 상시 녹화 또는 주차 중 이벤트 녹화를 설정하면 주차 중 녹화가 작동합니다.
  3. 수동 녹화가 필요할 때는 빌트인 캠 수동 녹화 버튼을 누릅니다.
  4. 수동 녹화 버튼을 짧게 누르면 이벤트 영상이 녹화됩니다.
  5. 수동 녹화 버튼을 길게 누르면 타임랩스 녹화가 시작되고, 다시 길게 누르면 종료됩니다.
  6. 녹화 영상을 확인하려면 인포테인먼트 시스템 홈 화면에서 빌트인 캠을 선택합니다.

해제 방법

  1. 주행/주차 중 녹화 설정을 끄면 해당 녹화가 작동하지 않습니다.
  2. 수동 녹화 중 시동을 끄거나 빌트인 캠 메뉴를 선택하면 녹화가 종료됩니다.
  3. 주차 중 녹화는 설정 시간이 초과되거나 12V 리튬 보조 배터리 충전량이 부족하면 종료됩니다.

작동 제한 상황

  1. SD 메모리가 제거된 상태에서는 녹화가 작동하지 않습니다.
  2. SD 메모리 수명이 다했거나 고장 난 경우 녹화할 수 없습니다.
  3. 빌트인 캠 메뉴 사용 중에는 녹화 기능이 작동하지 않습니다.
  4. 시동 후 빌트인 캠 시스템 부팅 중에는 영상 녹화가 되지 않습니다.
  5. 소프트웨어 버전에 따라 화면과 설명에 차이가 있을 수 있습니다.

주의 사항

  1. SD 메모리는 빌트인 캠 설정 > SD 메모리 메뉴의 연결 해제 버튼을 누른 뒤 제거해야 합니다.
  2. 중요한 영상은 자동 삭제될 수 있으므로 즉시 다운로드하거나 보관함에 저장하십시오.
  3. 음성 녹음 사용 시 탑승자에게 녹음 사실을 알리고 동의를 구하십시오.
  4. 전·후방 카메라 렌즈와 유리창이 오염되면 영상이 제대로 촬영되지 않을 수 있습니다.
  5. 빌트인 캠 Wi‑Fi 비밀번호가 타인에게 유출되지 않도록 주의하십시오.
  6. 제어기, 전·후방 카메라, 보조 배터리 커넥터를 강제로 분리하지 마십시오.
