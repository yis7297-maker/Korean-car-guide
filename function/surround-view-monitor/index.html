서라운드 뷰 모니터

서라운드 뷰 모니터 / SVM

차량 전방, 좌측방, 우측방, 후방 카메라 영상을 인포테인먼트 화면에 표시해 주차 또는 저속 주행을 돕는 기능입니다.

공식 검증 완료

기능 설명

차량 전방, 좌측방, 우측방, 후방 카메라 영상을 인포테인먼트 화면에 표시해 주차 또는 저속 주행을 돕는 기능입니다.

사용 전 조건

  • 서라운드 뷰 모니터용 카메라가 적용된 차량이어야 합니다.
  • 시동 'ON' 상태에서 표시 정보와 경고 방식을 설정할 수 있습니다.

설정 방법

  1. 설정 > 차량 > 운전자 보조 > 경고 방식 > 주차 안전 우선
  2. 서라운드 뷰 모니터 화면의 설정 버튼 또는 설정 > 운전자 보조 > 주차 안전 > 카메라 설정
  3. 표시 정보에서 탑 뷰 주차 가이드라인, 후방 주차 가이드라인, 주차 거리 경고를 설정합니다.

사용 방법

  1. 주차/뷰 버튼을 눌러 서라운드 뷰 모니터를 켜거나 끕니다.
  2. 'P', 'N', 'D' 상태에서 저속일 때 주차/뷰 버튼으로 전방 뷰 기능을 켤 수 있습니다.
  3. 설정에 따라 서라운드 뷰 모니터 자동 켜짐이 작동할 수 있습니다.
  4. 화면의 뷰 전환 버튼으로 탑 뷰, 전방 뷰, 사이드 뷰, 와이드 뷰 등을 선택합니다.

해제 방법

  1. 화면의 백 버튼을 누르거나 조건을 벗어나면 화면이 꺼질 수 있습니다.

제한 사항

  • 카메라 영상은 보정되어 표시되므로 실제 거리와 다를 수 있습니다.
  • 카메라 렌즈 오염이나 주변 환경에 따라 영상 품질이 저하될 수 있습니다.

주의 사항

  • 서라운드 뷰 모니터 화면만 의존하지 말고 주변을 직접 확인하십시오.
  • 렌즈 표면을 깨끗하게 유지하십시오.

FAQ

서라운드 뷰 모니터 / SVM 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
