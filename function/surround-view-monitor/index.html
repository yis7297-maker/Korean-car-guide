주차 편의

서라운드 뷰 모니터 / SVM

차량 주변 카메라 영상을 조합해 주차와 저속 이동 시 주변 확인을 돕습니다.

기능 개요

  1. 차량 주변 카메라 영상을 조합해 주차와 저속 이동 시 주변 확인을 돕습니다.

사용 전 조건

  1. 전방, 후방 및 사이드 미러 카메라가 가려져 있지 않아야 합니다.

설정 방법

  1. 주차/뷰 버튼을 누르거나 R(후진)로 전환해 화면을 실행합니다.

사용 방법

  1. 주차/뷰 버튼 또는 R(후진) 변속으로 영상을 표시합니다.
  2. 화면에서 확인할 뷰를 선택합니다.
  3. 영상과 실제 주변을 함께 확인하며 이동합니다.

해제 방법

  1. 주차/뷰 버튼을 누르거나 화면의 종료 버튼을 선택합니다.
  2. 차량 속도와 변속 조건에 따라 자동 종료될 수 있습니다.

작동 제한 상황

  1. 합성 영상의 물체 위치와 거리는 실제와 다르게 보일 수 있습니다.

주의 사항

  1. 카메라 영상과 경고는 주변 확인을 대신하지 않습니다. 운전자가 직접 차량 주변을 확인하고 제동과 조향을 수행해야 합니다.

관련 기능

  • 후방 모니터
  • 후측방 모니터 / BVM
