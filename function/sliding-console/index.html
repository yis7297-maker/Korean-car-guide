슬라이딩 콘솔

아이오닉 9의 슬라이딩 타입 중앙 콘솔박스를 앞뒤로 이동해 실내 공간 활용을 돕는 기능입니다.

기능 개요

  • 아이오닉 9의 슬라이딩 타입 중앙 콘솔박스를 앞뒤로 이동해 실내 공간 활용을 돕는 기능입니다.

사용 전 조건

  • 슬라이딩 콘솔 적용 사양이어야 합니다.

설정 방법

  • 별도 설정 메뉴가 아니라 콘솔의 레버를 사용합니다.

사용 방법

  • 레버를 당깁니다.
  • 콘솔박스를 앞 또는 뒤로 힘을 주어 밀거나 당깁니다.
  • 콘솔박스가 “딸깍” 하며 고정되는 소리가 나는지 확인합니다.

해제 방법

  • 원하는 위치에 고정되면 레버 조작을 멈춥니다.

작동 제한 상황

  • 과도한 힘으로 이동시키면 콘솔박스 레일 또는 레버가 손상될 수 있습니다.

주의 사항

  • 차가 움직이는 동안에는 슬라이딩 콘솔을 움직이지 마십시오. 급정차나 충돌 시 위험할 수 있습니다.

관련 기능

  • UV-C 살균 시스템
  • 100W USB 충전 단자
