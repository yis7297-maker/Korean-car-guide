원격 스마트 출차 보조

스마트 키 또는 지원 디지털 키 앱을 이용해 차량 밖에서 전진·후진 이동을 보조하는 원격 주차 보조 기능입니다.

기능 개요

  1. 스마트 키 또는 지원 디지털 키 앱을 이용해 차량 밖에서 전진·후진 이동을 보조하는 원격 주차 보조 기능입니다.

사용 전 조건

  1. 원격 스마트 주차 보조 사양이 적용된 차량이어야 합니다.
  2. 스마트 키 또는 지원되는 UWB 디지털 키를 휴대해야 합니다.
  3. 차량 주변에 사람, 동물, 물체가 없는지 확인해야 합니다.
  4. 차량과 키 또는 스마트폰이 작동 가능 거리 안에 있어야 합니다.

설정 방법

  1. 차량 실내에서는 주차/뷰 버튼을 길게 눌러 원격 스마트 주차 보조를 켭니다.
  2. 차량 밖에서는 스마트 키의 전진 또는 후진 버튼을 사용합니다.
  3. 디지털 키 사용 시 마이현대 앱의 원격 스마트 주차 보조 화면에서 원격 전/후진 화면으로 진입합니다.

사용 방법

  1. 전진 또는 후진하려는 공간 앞에 차량을 정렬하고 P(주차)로 변속합니다.
  2. 주차/뷰 버튼을 길게 눌러 원격 스마트 주차 보조를 켭니다.
  3. 스마트 키를 휴대하고 차량 밖으로 나와 모든 도어와 테일게이트를 닫습니다.
  4. 스마트 키의 전진 또는 후진 버튼을 누른 상태로 유지합니다.
  5. 차량이 원하는 위치에 도달하면 버튼에서 손을 떼거나 종료 버튼을 눌러 기능을 마칩니다.

해제 방법

  1. 스마트 키 버튼에서 손을 떼면 차량이 멈추고 제어가 일시 중지됩니다.
  2. 원격 시동 버튼 또는 종료 버튼을 누르면 기능이 종료될 수 있습니다.
  3. 원격 전/후진 완료 후 차량은 P(주차) 변속 및 전자식 파킹 브레이크 체결 상태로 전환될 수 있습니다.

작동 제한 상황

  1. 차량으로부터 약 4 m 이상 벗어나거나 다른 스마트 키 버튼 조작, 도어 열림, 도난 경보, 충돌방지 보조 작동 등이 발생하면 기능이 중지될 수 있습니다.
  2. 경사로, 눈길, 좁은 공간, 사선 주차, 센서 사각지대, 스토퍼, 트레일러 연결 등에서는 성능이 저하되거나 작동하지 않을 수 있습니다.

주의 사항

  1. 차량 외부에서 조작하는 동안에도 안전 책임은 운전자에게 있습니다.
  2. 차량 진행 방향 밖에서 조작하고 주변 사람·동물·물체를 계속 확인하십시오.
  3. 어린이가 스마트 키를 조작하지 않도록 주의하십시오.
