운전자 보조

스마트 크루즈 컨트롤 / SCC

전방 차량을 인식해 차간거리를 유지하고 운전자가 설정한 속도로 주행하도록 돕는 주행 편의 기능입니다.

공식 검증 완료

기능 설명

전방 차량을 인식해 차간거리를 유지하고 운전자가 설정한 속도로 주행하도록 돕는 주행 편의 기능입니다.

사용 전 조건

  • 기어가 D(주행)에 위치
  • 작동 속도 범위 내로 주행 중
  • 차체 자세 제어(ESC) 장치 및 ABS 장치가 켜진 상태

설정 방법

  1. 설정 > 차량 > 운전자 보조 > 주행 편의 > 스마트 크루즈 컨트롤에서 차간거리, 가속 강도, 반응 속도를 설정합니다.
  2. 사양 적용 시 설정 > 차량 > 운전자 보조 > 주행 편의 > 스마트 크루즈 컨트롤 > 운전 스타일 연동을 선택합니다.

사용 방법

  1. 주행 보조 버튼을 눌러 스마트 크루즈 컨트롤을 켭니다.
  2. 현재 클러스터 표시창의 속도로 설정속도가 지정됩니다.
  3. 차간거리 버튼을 반복해 차간거리 단계를 변경합니다.
  4. + 스위치를 올려 설정속도를 높이고, - 스위치를 내려 설정속도를 낮춥니다.
  5. 전방 차량이 있으면 설정한 차간거리를 유지하도록 속도를 조절합니다.

해제 방법

  1. 주행 보조 버튼 또는 브레이크 페달 등 매뉴얼에서 안내하는 조작으로 일시 해제하거나 끕니다.
  2. 운전 스타일 연동 중 차간거리 버튼을 길게 누르면 운전 스타일 연동이 비활성화됩니다.

제한 사항

  • 엔진이 고RPM으로 작동 중인 경우 작동하지 않을 수 있습니다.
  • ESC 또는 ABS가 작동 중이면 작동하지 않을 수 있습니다.
  • 전방 충돌방지 보조 또는 원격 스마트 주차 보조의 제동 제어가 작동 중이면 작동하지 않을 수 있습니다.

주의 사항

  • 추월 가속 보조 작동 중 위험 상황이 발생할 수 있으므로 항상 주의하십시오.
  • 주행 방향이 다른 국가에서 사용할 경우 위험할 수 있으므로 주의하십시오.
  • 전방 카메라와 레이더 관리를 유지하십시오.
