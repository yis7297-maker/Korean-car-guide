유틸리티 모드

고전압 배터리를 이용해 정차 중 차량 내부 전기 장치를 사용할 수 있도록 하는 EV 전용 모드입니다.

기능 개요

  1. 고전압 배터리를 이용해 정차 중 차량 내부 전기 장치를 사용할 수 있도록 하는 EV 전용 모드입니다.

사용 전 조건

  1. 주행 가능 표시등 또는 POWER ON 상태를 확인합니다.
  2. 변속 위치가 P(주차)에 있어야 합니다.
  3. 차량이 주행하지 않는 정차 상태여야 합니다.

설정 방법

  1. 인포테인먼트 시스템 홈 화면에서 EV(Electric Vehicle)를 선택합니다.
  2. EV 설정 메뉴에서 유틸리티 모드를 선택합니다.
  3. 유틸리티 모드 활성화 하기를 선택합니다.

사용 방법

  1. 작동 조건을 확인한 뒤 유틸리티 모드를 활성화합니다.
  2. UTIL 표시등이 켜지고 주차 브레이크가 체결되는지 확인합니다.
  3. 차량 내부 전기 장치를 필요한 범위에서 사용합니다.
  4. 실내 V2L을 함께 사용할 경우 차량 내부 전기 사용 안내를 따릅니다.

해제 방법

  1. 브레이크를 밟지 않고 시동 버튼을 눌러 OFF 상태로 전환합니다.
  2. 브레이크를 밟은 상태에서 시동 버튼을 눌러 DRIVE READY 상태로 전환합니다.
  3. EV 설정의 유틸리티 모드 활성화 항목을 다시 선택해 해제합니다.

작동 제한 상황

  1. 유틸리티 모드 사용 중에는 주행할 수 없습니다.
  2. P(주차) 이외의 변속 위치로 변경할 수 없습니다.
  3. 작동 조건을 만족해도 활성화되지 않으면 주차 브레이크 작동 상태를 점검해야 합니다.

주의사항

  1. 고전압 배터리 잔량과 차량 상태에 따라 사용 가능 시간이 달라질 수 있습니다.
  2. 차량 내부 전기 장치를 장시간 사용할 때는 배터리 잔량을 확인하십시오.

관련 기능

  1. 캠핑 모드
  2. 실내 V2L
