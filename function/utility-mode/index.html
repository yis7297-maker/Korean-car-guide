전동화

유틸리티 모드

전기차 특화 기능으로, EV 설정에서 유틸리티 모드를 활성화해 차량 전기 장치를 사용할 수 있는 기능입니다.

공식 검증 완료

기능 설명

전기차 특화 기능으로, EV 설정에서 유틸리티 모드를 활성화해 차량 전기 장치를 사용할 수 있는 기능입니다.

사용 전 조건

  • EV 설정 화면에 진입합니다.
  • 차량 상태가 유틸리티 모드 사용 조건을 만족합니다.

설정 방법

  1. EV(Electric Vehicle) > EV 설정 > 유틸리티 모드

사용 방법

  1. EV 설정에서 유틸리티 모드를 선택합니다.
  2. 유틸리티 모드를 활성화합니다.
  3. 차량 내부에서 전기 사용(V2L) 기능을 활용하려면 전기 사용(V2L) 관련 안내를 따릅니다.

해제 방법

  1. EV 설정에서 유틸리티 모드를 종료합니다.

제한 사항

  • 차량 상태와 배터리 상태에 따라 사용이 제한될 수 있습니다.

주의 사항

  • 유틸리티 모드 사용 중 배터리 상태를 확인하십시오.

FAQ

유틸리티 모드 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
