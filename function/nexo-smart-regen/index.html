스마트 회생제동 / 회생제동 조절

수소 전기차 감속 또는 내리막 주행 시 회생 제동으로 고전압 배터리를 충전해 주행 효율을 높입니다.

공식 검증 완료

기능 설명

수소 전기차 감속 또는 내리막 주행 시 회생 제동으로 고전압 배터리를 충전해 주행 효율을 높입니다.

사용 전 조건

  • 넥쏘 수소 전기차 주행 중입니다.

설정 방법

해당 차량의 공식 자료에서 별도 안내를 확인하세요.

사용 방법

  1. 감속 또는 내리막길 주행 시 회생 제동 브레이크가 작동합니다.
  2. 회생 제동으로 생성된 전기 에너지가 고전압 배터리에 저장됩니다.

해제 방법

해당 차량의 공식 자료에서 별도 안내를 확인하세요.

제한 사항

  • 차량 상태에 따라 회생 제동량이 달라질 수 있습니다.

주의 사항

  • 브레이크/회생 제동 경고 발생 시 점검을 받으십시오.

FAQ

스마트 회생제동 / 회생제동 조절 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
