FCEV / 회생제동

스마트 회생제동 / 회생제동 조절

주행 상황에 맞춰 감속 에너지를 회수하고 패들로 회생제동 단계를 조절합니다.

공식 자료 검증 완료

기능 설명

주행 상황에 맞춰 감속 에너지를 회수하고 패들로 회생제동 단계를 조절합니다.

사용 전 조건

  • 차량이 D(주행) 상태여야 합니다.
  • 회생제동 설정이 사용 가능해야 합니다.

설정 방법

  1. FCEV → 에너지 또는 수소전기차 설정

사용 방법

  1. 스티어링 휠 패들 시프트를 당겨 회생제동 단계를 조절합니다.
  2. 클러스터의 회생제동 단계 표시를 확인합니다.
  3. 스마트 회생제동은 차량 설정 또는 주행 보조 메뉴에서 켭니다.
  4. 앞차와 도로 상황에 따라 감속이 자연스러운지 확인합니다.

해제 방법

  1. 패들로 단계를 낮추거나 스마트 회생제동 설정을 끕니다.

제한 사항

  • 미끄러운 노면, 배터리 충전 상태, 급경사에서는 감속감이 달라질 수 있습니다.

주의 사항

  • 회생제동만으로 정차가 보장되지 않으므로 브레이크 페달을 준비하세요.
