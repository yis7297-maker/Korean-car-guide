지문 인증

등록된 지문으로 사용자를 인식해 스마트 키 없이 시동을 걸거나 일부 편의 기능을 사용할 수 있습니다.

공식 검증 완료

기능 설명

등록된 지문으로 사용자를 인식해 스마트 키 없이 시동을 걸거나 일부 편의 기능을 사용할 수 있습니다.

사용 전 조건

  • 인포테인먼트 시스템의 운전자 프로필에 지문이 등록되어 있어야 합니다.
  • 스마트 키 2개를 모두 차량 안에 둡니다.
  • 시동 ON 상태에서 등록을 진행합니다.

설정 방법

  1. 설정 > 사용자 프로필 > 운전자 1 또는 운전자 2 > 보안 > 지문인식

사용 방법

  1. 프로필 비밀번호를 입력합니다.
  2. 지문등록/삭제 > 등록을 선택합니다.
  3. 안내문에 따라 지문 인식 센서에 지문 여러 부위를 등록합니다.
  4. 지문 인증에 성공한 후 30초 안에 시동 버튼을 눌러 시동을 겁니다.

해제 방법

  1. 설정 > 사용자 프로필 > 운전자 1 또는 운전자 2 > 보안 > 지문인식에서 지문등록/삭제 > 삭제를 선택합니다.

제한 사항

  • 지문 등록 중 인포테인먼트 화면 변경 또는 시동 버튼 사용 시 등록이 중단될 수 있습니다.
  • 운전자 1과 운전자 2에 동일한 지문을 등록할 수 없습니다.

주의 사항

  • 복제한 지문도 시동이 걸릴 수 있으므로 다른 사람이 지문을 복제하지 못하게 하십시오.
  • 일부 기능은 블루링크 서비스 가입이 필요할 수 있습니다.
