기능 설명

등록된 지문으로 운전자 프로필 호출이나 시동 인증을 보조합니다.

사용 전 조건

  • 지문 인증 사양이 탑재되어야 합니다.
  • 차량 전원 ON 상태에서 사용자 프로필에 접근할 수 있어야 합니다.

설정 방법

  1. 설정 → 차량 → 디지털 키 또는 사용자 인증

사용 방법

  1. 차량 설정 > 사용자 프로필 또는 보안 메뉴를 엽니다.
  2. 지문 등록을 선택하고 센서에 손가락을 여러 번 올립니다.
  3. 등록 완료 후 프로필 연동 또는 시동 인증 사용 여부를 선택합니다.

해제 방법

  1. 사용자 프로필/보안 메뉴에서 등록된 지문을 삭제합니다.

제한 사항

  • 손이 젖었거나 센서가 오염되면 인식률이 낮아질 수 있습니다.

주의 사항

  • 공유 차량에서는 등록 지문과 프로필 권한을 주기적으로 점검하세요.

관련 기능

연결된 관련 기능이 없습니다.
