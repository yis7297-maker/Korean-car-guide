후진 연동 사이드 미러

후진 시 사이드 미러 각도를 주차 확인 위치로 조절하고 후진 종료 후 원래 위치로 복귀합니다.

기능 개요

  1. 후진 시 사이드 미러 각도를 주차 확인 위치로 조절하고 후진 종료 후 원래 위치로 복귀합니다.

사용 전 조건

  1. 사이드 미러 선택 스위치가 차량 매뉴얼에 안내된 작동 위치에 있어야 합니다.

설정 방법

  1. 사이드 미러 선택 스위치와 미러 조절 스위치로 후진 위치를 설정합니다.

사용 방법

  1. R(후진)로 전환해 미러가 내려가는지 확인합니다.
  2. 필요한 경우 미러 조절 스위치로 후진 위치를 조절합니다.

해제 방법

  1. 사이드 미러 선택 스위치를 중립 위치로 두면 기능이 작동하지 않습니다.
  2. R(후진) 이외의 위치로 변속하면 미러가 복귀합니다.

작동 제한 상황

  1. 작동 및 저장 방식은 차량 사양에 따라 다릅니다.

주의 사항

  1. 미러에 보이는 거리와 실제 거리가 다를 수 있으므로 직접 주변을 확인하세요.

  3. 제네시스 공식 다운로드 센터 1순위-4순위
