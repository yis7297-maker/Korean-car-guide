운전자 보조

지능형 속도 제한 보조 / ISLA

도로 표지판과 내비게이션 정보를 활용해 제한속도와 부가 표지판 정보를 알려주고 제한속도 준수를 돕습니다.

공식 검증 완료

기능 설명

도로 표지판과 내비게이션 정보를 활용해 제한속도와 부가 표지판 정보를 알려주고 제한속도 준수를 돕습니다.

사용 전 조건

  • 시동 ON 상태
  • 전방 카메라 및 내비게이션 도로 정보가 정상 작동하는 상태
  • 속도 제한 기능 설정이 켜진 상태

설정 방법

  1. 설정 > 차량 > 운전자 보조 > 속도 제한에서 기능 사용 여부를 설정합니다.
  2. 속도 오프셋 설정, 속도 제한 보조, 속도 제한 경고, 끄기 중 선택합니다.

사용 방법

  1. 클러스터 표시창에 제한속도 정보를 표시합니다.
  2. 표시된 제한속도보다 빠르게 주행하면 과속 경고를 제공합니다.
  3. 수동 속도 제한 보조 또는 스마트 크루즈 컨트롤 작동 중 제한속도가 변경되면 + 또는 - 스위치로 설정속도를 변경할 수 있습니다.
  4. 조건이 맞으면 제한속도 변경에 맞춰 자동으로 설정속도를 조절할 수 있습니다.

해제 방법

  1. 설정 > 차량 > 운전자 보조 > 속도 제한에서 끄기를 선택합니다.

제한 사항

  • 표지판 인식이 어려운 경우 제한속도 정보가 표시되지 않거나 잘못 인식될 수 있습니다.
  • 전방 카메라 앞 유리 이물질로 일시 제한될 수 있습니다.
  • 도로의 제한속도가 30 km/h 미만인 경우 설정속도 변경 기능은 작동하지 않습니다.

주의 사항

  • 반드시 안전한 곳에 정차한 후 설정하십시오.
  • 항상 교통 상황을 확인하고 필요한 경우 운전자가 직접 감속해야 합니다.
  • 내비게이션 정기 업데이트가 필요합니다.
