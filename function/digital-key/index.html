디지털 키/인증

Hyundai Digital Key

스마트폰 또는 카드 키를 이용해 도어 잠금/잠금 해제와 시동을 보조하는 현대 디지털 키 기능입니다.

기능 개요

  • 스마트폰 또는 카드 키를 이용해 도어 잠금/잠금 해제와 시동을 보조하는 현대 디지털 키 기능입니다.

사용 전 조건

  • 디지털 키 적용 사양이어야 하며, 스마트폰 디지털 키 또는 카드 키 등록이 필요합니다.

설정 방법

  • 인포테인먼트 시스템의 설정 > 차량 > 디지털 키 메뉴를 사용합니다.

사용 방법

  • 스마트폰 또는 카드 키를 등록합니다.
  • 도어 핸들 또는 실내 인증 패드에 디지털 키를 태그해 도어 잠금/잠금 해제 또는 시동 기능을 사용합니다.

해제 방법

  • 디지털 키 메뉴에서 등록된 키를 삭제하거나 사용 설정을 해제합니다.

작동 제한 상황

  • 차량 시스템에 등록된 카드 키가 있으면 새 카드 키를 등록할 수 없습니다.

주의 사항

  • 디지털 키 등록 시 스마트 키 2개를 휴대합니다.
  • 차량에 등록한 카드 키는 다른 차량에서 등록할 수 없습니다.

관련 기능

  • NFC 카드 키
  • 스마트 키
