기능 설명

패들 또는 차량 설정으로 하이브리드 회생제동의 감속감과 에너지 회수 수준을 조절합니다.

사용 전 조건

  • TMED-2 또는 해당 하이브리드 전용 기능이 적용된 차량이어야 합니다.

설정 방법

해당 차량의 공식 자료에서 별도 안내를 확인하세요.

사용 방법

  1. 차량 화면에서 하이브리드 또는 해당 기능 메뉴를 선택합니다.
  2. 차량 상태와 화면 안내를 확인한 뒤 기능을 사용합니다.

해제 방법

  1. 기능 화면에서 종료를 선택하거나 차량 전원 상태를 변경합니다.

제한 사항

  • 고전압 배터리 충전량, 전력 사용량, 외기 온도와 차량 상태에 따라 작동 범위가 달라질 수 있습니다.

주의 사항

  • 밀폐된 공간에서는 엔진이 자동으로 작동할 가능성에 대비해 환기와 안전을 확인하세요.
