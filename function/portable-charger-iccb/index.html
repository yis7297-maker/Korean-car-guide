휴대용 충전기(ICCB)

코나 일렉트릭 매뉴얼에 수록된 휴대용 충전기(ICCB)를 이용한 충전 기능입니다.

기능 개요

  • 코나 일렉트릭 매뉴얼에 수록된 휴대용 충전기(ICCB)를 이용한 충전 기능입니다.

사용 전 조건

  • 휴대용 충전기(ICCB)를 사용할 수 있는 전원 환경과 차량 충전 조건을 만족해야 합니다.

설정 방법

  • 충전 전류 및 충전 상태를 확인합니다.

사용 방법

  • 휴대용 충전기(ICCB) 사용하기 절차에 따라 전원과 차량을 연결합니다.
  • 충전 상태 표시 정보를 확인합니다.
  • 충전 완료 후 매뉴얼 안내에 따라 분리합니다.

해제 방법

  • 충전 완료 또는 중단 후 충전 장치를 분리합니다.

작동 제한 상황

  • 전원 환경과 충전 장치 상태에 따라 충전 시간이 길어질 수 있습니다.

주의 사항

  • 정격에 맞지 않는 전원이나 손상된 충전 장치를 사용하지 않습니다.

관련 기능

  • 완속 충전
  • 충전 상태 확인
