EV 충전 예약 / 배터리 컨디셔닝 / V2L

전기차 충전 시간과 목표량을 관리하고 급속 충전 준비 및 외부 전원 공급을 지원합니다.

공식 자료 검증 완료

기능 설명

전기차 충전 시간과 목표량을 관리하고 급속 충전 준비 및 외부 전원 공급을 지원합니다.

사용 전 조건

  • 전동화 차량이어야 합니다.
  • 충전 케이블 또는 V2L 커넥터 상태가 정상이어야 합니다.

설정 방법

  1. EV → 전력 공급

사용 방법

  1. EV 메뉴에서 충전 설정을 엽니다.
  2. 목표 충전량과 예약 시간을 선택합니다.
  3. 배터리 컨디셔닝은 내비게이션 목적지를 급속 충전소로 설정한 뒤 조건을 확인합니다.
  4. V2L은 커넥터를 연결하고 사용 가능 전력을 확인합니다.

해제 방법

  1. 예약 충전은 EV 메뉴에서 예약을 끕니다.
  2. V2L은 전원 공급 중지를 누르고 커넥터를 분리합니다.

제한 사항

  • 외부 온도, 충전기 상태, 배터리 잔량에 따라 제한됩니다.

주의 사항

  • 젖은 손이나 손상된 케이블로 충전하지 마세요.
