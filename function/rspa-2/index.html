주차 편의

원격 스마트 주차 보조 2 / RSPA 2

차량 밖에서 전/후진 또는 후진 직각·평행 주차를 돕고, 차량 안에서도 후진 직각·평행 주차 또는 평행 출차를 도와주는 기능입니다.

공식 검증 완료

기능 설명

차량 밖에서 전/후진 또는 후진 직각·평행 주차를 돕고, 차량 안에서도 후진 직각·평행 주차 또는 평행 출차를 도와주는 기능입니다.

사용 전 조건

  • 차량 사양 또는 파킹 어시스트 옵션 적용 여부를 확인해야 합니다.
  • 스마트 키 또는 지원되는 UWB 디지털 키를 사용합니다.
  • 버튼 조작 중에는 주변 상황을 계속 확인해야 합니다.

설정 방법

  1. 블루링크 스토어의 디지털 사양 > 원격 스마트 주차 보조 2 구매가 필요한 차량이 있을 수 있습니다.
  2. 주차/뷰 버튼으로 기능을 시작합니다.

사용 방법

  1. 주차/뷰 버튼을 길게 눌러 원격 스마트 주차 보조, 주차 거리 경고, 서라운드 뷰 모니터를 켭니다.
  2. 스마트 주차 또는 스마트 출차 작동 중에는 주차/뷰 버튼을 누르고 있어 주차 또는 출차를 시작합니다.
  3. 스마트 키의 전진 버튼 또는 후진 버튼을 누르고 있으면 해당 방향으로 원격 전/후진 또는 원격 주차가 작동합니다.
  4. UWB 디지털 키 사용 시 마이현대 앱에서 원격 스마트 주차 보조 버튼을 누르고 사용자 동의 후 원격 스마트 주차 보조 화면으로 이동합니다.
  5. 원격 전/후진은 차량 아이콘을 누르고 밀어 해당 방향으로 차량을 이동합니다.
  6. 원격 주차는 아이콘을 밀고 누르고 있는 동안 기능이 작동합니다.

해제 방법

  1. 조작 버튼에서 손을 떼면 진행 중인 원격 제어가 중지됩니다.
  2. 상황별 안내에 따라 기능을 종료합니다.

제한 사항

  • UWB 디지털 키 기능은 UWB 지원 차량과 스마트폰에서만 작동합니다.
  • 사양 기본 적용 차량 외에는 파킹 어시스트 옵션 및 디지털 사양 구매가 필요할 수 있습니다.

주의 사항

  • 버튼 조작 중에는 항상 주변 상황을 확인해야 합니다.
  • 차량 외부에서 조작하더라도 운전자가 안전 책임을 가지고 주변을 확인해야 합니다.

FAQ

원격 스마트 주차 보조 2 / RSPA 2 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
