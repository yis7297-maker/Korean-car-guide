수소 탱크 정보

FCEV 모드에서 수소 탱크의 온도, 압력, 충전 상태를 확인하는 기능입니다.

기능 개요

  • FCEV 모드에서 수소 탱크의 온도, 압력, 충전 상태를 확인하는 기능입니다.

사용 전 조건

  • 차량 전원이 켜져 있고 인포테인먼트 시스템에서 FCEV 메뉴를 사용할 수 있어야 합니다.

설정 방법

  • 홈 화면에서 FCEV 메뉴를 선택합니다.

사용 방법

  • 인포테인먼트 홈 화면에서 FCEV를 선택합니다.
  • 수소 탱크 정보 화면에서 탱크 온도, 압력, 충전 상태를 확인합니다.

해제 방법

  • 다른 화면으로 이동하거나 인포테인먼트 시스템을 종료합니다.

작동 제한 상황

  • 표시 정보는 차량 상태와 센서 상태에 따라 달라질 수 있습니다.

주의 사항

  • 수소 계통 경고 또는 이상 표시가 있으면 차량을 안전한 곳에 정차하고 제조사 안내에 따릅니다.

관련 기능

  • FCEV 정보 화면 / 에너지 모니터
  • 수소 충전 / 충전구 개폐
  • 수소 주행 가능 거리
