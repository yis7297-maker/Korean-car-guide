플레오스 OTA / 앱 연동

더 뉴 그랜저 플레오스 기반의 OTA와 차량 앱 연동 후보 기능입니다.

기능 개요

  1. 더 뉴 그랜저 플레오스 기반의 OTA와 차량 앱 연동 후보 기능입니다.

사용 전 조건

  1. 플레오스 적용 사양, 네트워크, 계정 조건이 필요할 수 있습니다.

사용 방법

  1. 플레오스 앱 또는 업데이트 메뉴를 엽니다.
  2. 사용 가능한 앱/업데이트 목록을 확인합니다.
  3. 정차 상태에서 설치 또는 업데이트를 실행합니다.

해제 방법

  1. 앱 메뉴에서 삭제/비활성화하거나 업데이트를 나중에 하기로 선택합니다.

작동 제한 상황

  1. 공식 기능 범위와 앱 목록 재확인이 필요합니다.

주의 사항

  1. 업데이트 중에는 안내 없이 전원을 끄지 마세요.

관련 기능
