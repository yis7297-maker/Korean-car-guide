주차 충돌방지 보조

주차 또는 저속 이동 중 충돌 위험을 감지해 경고하고 기능 조건에 따라 긴급 제동을 보조합니다.

기능 개요

  1. 주차 또는 저속 이동 중 충돌 위험을 감지해 경고하고 기능 조건에 따라 긴급 제동을 보조합니다.

주의 사항

  1. 카메라 영상과 경고는 주변 확인을 대신하지 않습니다. 운전자가 직접 차량 주변을 확인하고 제동과 조향을 수행해야 합니다.

관련 기능

  • 측방 주차 거리 경고
  • 메모리 리버스 어시스트 / MRA
