UV-C 살균 시스템

아이오닉 9의 보관함 내부 UV-C LED를 이용해 보관한 물건을 일정 시간 살균하는 편의 기능입니다.

기능 개요

  • 아이오닉 9의 보관함 내부 UV-C LED를 이용해 보관한 물건을 일정 시간 살균하는 편의 기능입니다.

사용 전 조건

  • 시동이 켜져 있어야 합니다.
  • UV-C 살균 시스템 적용 사양이어야 합니다.

설정 방법

  • 보관함 내부의 UV-C 살균 시스템 작동 버튼을 사용합니다.

사용 방법

  • 보관함 커버를 엽니다.
  • 보관함 내부의 UV-C 살균 시스템 작동 버튼을 누릅니다.
  • 살균할 물건을 보관함 내부 중앙에 놓고 커버를 닫습니다.
  • 살균 작동 표시등이 켜지고 UV-C LED가 켜지며 살균이 시작됩니다.
  • 살균 시작 10분 후 살균 작동 표시등이 자동으로 꺼지며 살균이 종료됩니다.

해제 방법

  • 10분 이내에 직접 종료할 경우 UV-C 살균 시스템 작동 버튼을 눌러 기능을 끕니다.

작동 제한 상황

  • 살균 트레이가 작동 가능한 상태여야 합니다.

주의 사항

  • UV-C 살균 시스템 사용 중에는 매뉴얼의 안전 주의사항을 준수합니다.

관련 기능

  • 스마트폰 무선 충전 시스템
  • 슬라이딩 콘솔
