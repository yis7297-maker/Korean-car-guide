V2L

구동용 고전압 배터리 전력을 외부 전기 기기에 공급하는 전기 사용 기능입니다.

공식 검증 완료

기능 설명

구동용 고전압 배터리 전력을 외부 전기 기기에 공급하는 전기 사용 기능입니다.

사용 전 조건

  • V2L 지원 차량입니다.
  • 구동용 고전압 배터리 충전량이 방전 제한량보다 높습니다.
  • V2L 커넥터와 충전 인렛, 전원 플러그가 정상 상태입니다.

설정 방법

  1. EV(Electric Vehicle) > 전기 사용(V2L) 설정

사용 방법

  1. 전기 사용(V2L) 설정에서 방전 제한량을 설정합니다.
  2. V2L 커넥터 또는 차량 전원 패널에 전기 기기를 연결합니다.
  3. 사용 후 V2L 커넥터 스위치를 눌러 전원을 끄고 차량 도어 잠금을 해제한 뒤 커넥터를 분리합니다.

해제 방법

  1. 배터리 충전량이 설정된 방전 제한량에 도달하면 자동 차단됩니다.
  2. 사용 후 전원을 끄고 커넥터를 분리합니다.

제한 사항

  • V2L 커넥터 내부 과열 시 자동 차단됩니다.
  • 커넥터, 인렛, 플러그, 케이블 손상 또는 부식 시 사용하지 마십시오.

주의 사항

  • 젖은 손으로 V2L 커넥터나 충전 인렛, 전원 플러그를 만지지 마십시오.
  • V2L 커넥터를 임의로 개조하거나 분해하지 마십시오.
  • V2L 커넥터로 차량을 충전하지 마십시오.

FAQ

V2L 기능의 적용 여부는 어떻게 확인하나요?

현재 차종별 적용 정보가 공식 원문 기준으로 확정되지 않았습니다. 해당 차량의 공식 매뉴얼과 제조사 안내를 확인하세요.
