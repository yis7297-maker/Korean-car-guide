클러스터와 내비게이션 화면을 넓게 연결해 주행 정보와 인포테인먼트를 통합 표시합니다.

공식 자료 검증 완료

기능 설명

클러스터와 내비게이션 화면을 넓게 연결해 주행 정보와 인포테인먼트를 통합 표시합니다.

사용 전 조건

  • 해당 디스플레이 사양 탑재 차량이어야 합니다.

설정 방법

  1. 홈 → 설정 → 차량 → 해당 기능 메뉴

사용 방법

  1. 시동을 켜고 클러스터와 내비게이션 화면 부팅을 확인합니다.
  2. 홈 화면에서 내비게이션, 미디어, 차량 설정을 선택합니다.
  3. 스티어링 휠 또는 화면 설정에서 표시 테마를 조정합니다.

해제 방법

  1. 기능 자체는 끌 수 없으며 화면 밝기/표시 항목만 조정합니다.

제한 사항

  • 일부 앱과 표시 항목은 주행 중 제한됩니다.

주의 사항

  • 주행 중 터치 조작은 최소화하세요.

관련 기능

연결된 관련 기능이 없습니다.
