원격 공조

공식 앱에서 탑승 전 실내 온도, 열선, 통풍 등 일부 공조 기능을 미리 실행합니다.

공식 자료 검증 완료

기능 설명

공식 앱에서 탑승 전 실내 온도, 열선, 통풍 등 일부 공조 기능을 미리 실행합니다.

사용 전 조건

  • Bluelink/Kia Connect/Genesis Connected Services 가입과 앱 로그인이 필요합니다.
  • 차량 통신과 배터리 상태가 정상이어야 합니다.
  • 차량이 잠겨 있고 안전 조건을 만족해야 합니다.

설정 방법

  1. 공조 화면 → 공조 설정

사용 방법

  1. 공식 앱에서 차량을 선택합니다.
  2. 원격 공조 또는 원격 시동 메뉴를 누릅니다.
  3. 희망 온도와 열선/통풍 옵션을 선택합니다.
  4. 앱 보안 인증을 완료하고 실행 결과를 확인합니다.

해제 방법

  1. 앱에서 원격 공조 중지를 누릅니다.
  2. 차량에 탑승해 시동/전원 상태를 변경합니다.

제한 사항

  • 지하 주차장, 통신 음영, 도어 열림, 변속 상태, 배터리 부족 시 실패할 수 있습니다.

주의 사항

  • 밀폐된 장소에서 장시간 원격 시동/공조를 사용하지 마세요.

관련 기능

연결된 관련 기능이 없습니다.
